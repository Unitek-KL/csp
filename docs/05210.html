Library Interface
a) Initialization Function
 
Name 
Description 
 
PMP_Initialize 
This function initializes the PMP controller of the device. 
b) Setup Functions
 
Name 
Description 
 
PMP_AddressGet 
Gets the current address of the PMP module. 
 
PMP_AddressPortDisable 
Disables the port lines specified as PMP address lines. 
 
PMP_AddressPortEnable 
Enables the port lines specified as PMP address lines. 
 
PMP_AddressSet 
Sets the current address of the PMP module to the specified address. 
c) Transaction Functions
 
Name 
Description 
 
PMP_MasterReceive 
Receives the data in Master mode. 
 
PMP_MasterSend 
Sends the specified data in Master mode. 
d) Status Functions
 
Name 
Description 
 
PMP_PortIsBusy 
Identifies if the (Master mode) PMP port is busy.
