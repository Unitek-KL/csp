USARTx_ReceiverIsReady Function

This function returns the hardware status associated with the Receive hardware FIFO of the given USART peripheral instance. 

When Receiver is ready, user can read the available data. 

This function is available only in non-interrupt mode of operation and it should be used to check receiver readiness before calling USART1_ReadByte function.

C
bool USARTx_ReceiverIsReady();
Preconditions

USARTx_Initialize must have been called for the associated USART instance.

Returns
  • true - Receive hardware FIFO has at least one data available to read.
  • false - Receive hardware FIFO is empty.
Remarks

None.

Example
char rxData;
if(true == USART1_ReceiverIsReady())
{
    // At least one data is available in Receive FIFO, hence can read a byte
    rxData = USART1_ReadByte();
}
else
{
    //Receiver is empty
}
