EEPROM_WordWrite Function

Writes one word to data EEPROM.

C
bool EEPROM_WordWrite(
    uint32_t addr, 
    uint32_t data
);
Preconditions

None

Returns

Returns true on success false on error

Example
uint32_t i;
uint32_t writeData[4];
uint32_t addr = 0;

for ( i = 0; i < 4; i++ )
{
    result = EEPROM_WordWrite( addr, writeData[i] );
    if (result == true)
    {
        addr = addr + 4;
    }
}
