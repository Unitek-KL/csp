Running The Application

Running project with MPLAB X IDE 

  1. Build and Program the application using the MPLAB X IDE.
  2. Observe the Output as described in the following table:
Project Name 
Output 
pic32mk_gp_db.X 
Observe the clock of 4 MHz on pin RPA0 (Pin #4 on the J29 on the PIC32MK General Purpose (GP) Development Board). 
pic32mx470_curiosity.X 
Observe the clock of 4 MHz on pin RPB8 (Pin #14 on the J14 on the PIC32MX Curiosity Development Board. 
pic32mx_xlp_sk.X 
Observe the clock of 4 MHz on pin RPC6 (Pin #3 on the J2 on the PIC32MX XLP Starter Kit. 
pic32mz_ef_sk.X 
Observe the clock of 4 MHz on pin PA14 (Pin #7 on the J12 on the PIC32MZ Embedded Connectivity with FPU (EF) Starter Kit). 
sam_c21n_xpro.X 
Observe the clock of 4 MHz on pin PB23 (Pin #8 on the EXT3 on the SAM C21N Xplained Pro Evaluation Kit). 
sam_d20_xpro.X 
Observe the clock of 4.8 MHz on pin PB15 (Pin #10 on the EXT2 header on the SAM D20 Xplained Pro Evaluation Kit. 
sam_d21_xpro.X 
Observe the clock of 4.8 MHz on pin PB15 (Pin #10 on the EXT2 header on the SAM D21 Xplained Pro Evaluation Kit). 
sam_e54_xpro.X 
Observe the clock of 5 MHz on pin PB15 (Pin #8 on the EXT2 header on the SAM E54 Xplained Pro Evaluation Kit). 
sam_e70_xult.X 
Observe the clock of 5 MHz on pin PA03 (Pin #11 on the EXT1 header on the SAM V71 Xplained Ultra Evaluation Kit). 
sam_l21_xpro.X 
Observe the clock of 4 MHz on pin PB15 (Pin #10 on the EXT2 header on the SAM L21 Xplained Pro Evaluation Kit). 
sam_l22_xpro.X 
Observe the clock of 1 MHz on pin PA15 (Pin #13 on the EXT3 header on the SAM L22 Xplained Pro Evaluation Kit). 
sam_v71_xpro.X 
Observe the clock of 5 MHz on pin PA03 (Pin #11 on the EXT1 header on the SAM V71 Xplained Ultra Evaluation Kit). 

 

Running project with IAR Embedded work bench

  1. Build the project using IAR Embedded Workbench
  2. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  3. Reset the board to run the application

 

Project Name 
Output 
sam_a5d2_xult.IAR 
Observe the clock of 4.15 MHz on pin PD19 (Pin #2 on the J8 Connector on SAMA5D2 Xplained Ultra Evaluation Kit). 
sam_9x60_ek.IAR 
Observe the clock of 5 MHz on pin PC31 (Pin #29 on the J17 Connector on SAM9X60 evaluation kit).
