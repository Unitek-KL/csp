FLEXCOMx_USART_SerialSetup Function

This function sets the FLEXCOM USART serial communication settings dynamically.

C
bool FLEXCOMx_USART_SerialSetup(
    FLEXCOM_USART_SERIAL_SETUP * setup, 
    uint32_t srcClkFreq
);
Preconditions

FLEXCOMx_USART_Initialize must have been called for the associated FLEXCOM USART instance. The FLEXCOM USART transmit or receive transfer status should not be busy.

Parameters
Parameters 
Description 
setup 
Pointer to the structure containing the serial setup. 
srcClkFreq 
FLEXCOM USART Peripheral Clock Source Frequency. 
Returns

true - Serial setup was updated Successfully. false - Failure while updating serial setup.

Remarks

srcClkFreq overrides any change in the peripheral clock frequency. If configured to zero PLib takes the peripheral clock frequency from MHC.

Example
FLEXCOM_USART_SERIAL_SETUP setup = {
        115200,
        FLEXCOM_USART_DATA_8_BIT,
        FLEXCOM_USART_PARITY_ODD,
        FLEXCOM_USART_STOP_1_BIT
    };

FLEXCOMx_USART_SerialSetup(&setup, 0);
