SDMMCx_IsCardAttached Function

Indicates if the card is attached or detached.

C
bool SDMMCx_IsCardAttached();
Preconditions

SDMMCx_Initialize() must have been called first for the associated instance.

Returns

true - If the card is inserted. false - If the card is removed.

Remarks

None.

Example
if (SDMMC1_IsCardAttached() == true)
{
    // Card is attached
}
