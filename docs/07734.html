Running The Application
  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build and program the application using the MPLAB X IDE
  4. Console will print the value contained in the TRAM. Observe that the TRAM contains non zero values.

  1. To emulate tampering, create a falling edge on the RTC input pin. This is achieved as follows:

SAM L10 Xplained Pro Evaluation Kit 

  • PA08 pad is the RTC input 0 and is available on connector EXT1, pin 14
  • Vcc is available on connector J100, pin 4
  • Short PA08 to Vcc and then remove the short to create a falling edge

 

 

  1. Console will output the current value of TRAM. Observe that the TRAM values now read zero indicating a erasure of the TRAM content.
