TMR1_CounterGet Function

This function reads the timer counter value.

C
uint16_t TMR1_CounterGet();
Preconditions

TMR1_Initialize() function must have been called first.

Returns

counter - The timer's current count value.

Remarks

None

Example
uint16_t counter;

TMR1_Initialize();
TMR1_Start();
counter = TMR1_CounterGet();
