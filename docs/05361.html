SDMMC_DataTransferFlags Structure

SDMMC data transfer flags 

The client indicates whether the transfer contains data stage and if yes, it further specifies the data transfer direction and data transfer type. The object of this data type is passed as an argument to the SDMMCx_CommandSend function.

C
typedef struct {
  bool isDataPresent;
  SDMMC_DATA_TRANSFER_DIR transferDir;
  SDMMC_DATA_TRANSFER_TYPE transferType;
} SDMMC_DataTransferFlags;
Members
Members 
Description 
bool isDataPresent; 
Indicates if SDMMC transfer has a data stage or not 
SDMMC_DATA_TRANSFER_DIR transferDir; 
Direction of the data transfer 
SDMMC_DATA_TRANSFER_TYPE transferType; 
Type of the data transfer - read/write 
Remarks

None.
