AFEC_CHANNEL_NUM Enumeration

Analog channel numbers 

This enumeration identifies AFEC channel numbers. This may be used as argument of function to identify channel number.

C
typedef enum {
  AFEC_CH0,
  AFEC_CH1,
  AFEC_CH2,
  AFEC_CH3,
  AFEC_CH4,
  AFEC_CH5,
  AFEC_CH6,
  AFEC_CH7,
  AFEC_CH8,
  AFEC_CH9,
  AFEC_CH10,
  AFEC_CH11
} AFEC_CHANNEL_NUM;
Remarks

None.
