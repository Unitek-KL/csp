Microchip Support
Library Interface
a) Setup Functions
 
Name 
Description 
 
NVM_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the operation is complete. 
b) Transaction Functions
 
Name 
Description 
 
NVM_Read 
Reads length number of bytes from a given address in FLASH memory. 
 
NVM_PageErase 
Erases a Page in the NVM 
 
NVM_RowWrite 
Writes one row of data to given NVM address. 
 
NVM_WordWrite 
Writes One word into the Flash. 
 
NVM_QuadWordWrite 
Writes four words into the Flash. 
 
NVM_BootFlashWriteProtectDisable 
Un-Protect Boot Flash Memory Page to perform Writes 
 
NVM_BootFlashWriteProtectEnable 
Protect Boot Flash Memory Page from Writes 
 
NVM_BootFlashWriteProtectLock 
Disable Writes to Boot flash Write Protect Lock register 
 
NVM_ProgramFlashSwapBank 
Swaps the Program Flash Bank 2 
 
NVM_ProgramFlashWriteProtect 
Protect Program Flash Memory from Writes 
 
NVM_ProgramFlashWriteProtectLock 
Disable Writes to Program Flash Write Protect Lock register 
c) Status Functions
 
Name 
Description 
 
NVM_IsBusy 
Returns the current status of NVM controller. 
 
NVM_ErrorGet 
Returns the error state of NVM controller. 
d) Data Types and Constants
 
Name 
Description 
 
NVM_ERROR 
Defines the NVM Error Type. 
 
NVM_CALLBACK 
Defines the data type and function signature for the NVM peripheral callback function. 
 
NVM_BOOT_FLASH_WRITE_PROTECT 
Defines the Boot flash write protect pages. 
 
NVM_BOOT_FLASH_WRITE_PROTECT_LOCK 
Defines the Write protect lock pages.
