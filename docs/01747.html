FLEXCOMx_TWI_Read Function

This function reads the data of size in bytes equal to length into pdata buffer from the slave having given address. Master will generate Stop condition after completion of the read.

C
bool FLEXCOMx_TWI_Read(
    uint16_t address, 
    uint8_t * pdata, 
    size_t length
);
Preconditions

FLEXCOMx_TWI_Initialize must have been called for the associated FLEXCOM TWI instance.

Parameters
Parameters 
Description 
address 
7-bit / 10-bit slave address. 
pdata 
pointer to destination data buffer 
length 
length of data buffer in number of bytes. 
Returns

Read request status. True - Read request was successful. False - Read request has failed.

Remarks

None.

Example
    uint8_t myData [NUM_BYTES];
  
    if(!FLEXCOMx_TWI_Read( SLAVE_ADDR, &myData[0], NUM_BYTES ))
    {
        // error handling
    }
