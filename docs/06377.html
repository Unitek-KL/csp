Running The Application
  1. Build and Program the application using the MPLAB X IDE.
  2. Observe the output as described in the following table:
Project Name 
Output 
sam_c21n_xpro.X 
Observe the high-side and low-side PWM waveforms on oscilloscope. Observe the dead time between the high side and the low side. PWM frequency = 10KHz

  • TCC Channel 2
    • Pin 5 of EXT1 - PWMH (PA10)
    • Pin 7 of EXT1 - PWML ( PB12)
  • TCC Channel 3
    • Pin 6 of EXT1 - PWMH ( PA11)
    • Pin 8 of EXT1 - PWML ( PB13)


 

sam_d21_xpro.X 
Observe the high-side and low-side PWM waveforms on oscilloscope. Observe the dead time between the high side and the low side. PWM frequency = 10KHz

  • TCC Channel 0
    • Pin 11 of EXT2 - PWMH (PA08)
    • Pin 14 of EXT2 - PWML ( PB10)
  • TCC Channel 1
    • Pin 12 of EXT2 - PWMH ( PA09)
    • Pin 13 of EXT2 - PWML ( PB11)
  • TCC Channel 2
    • Pin 3 of EXT2 - PWMH ( PA10)
    • Pin 7 of EXT2 - PWML ( PB12)


 

sam_e54_xpro.X 
Observe the high-side and low-side PWM waveforms on oscilloscope. Observe the dead time between the high side and the low side. PWM frequency = 10KHz

  • TCC Channel 0
    • Pin 16 of EXT2 - PWMH (PC04)
    • Pin 8 of EXT3 - PWML ( PD11)
  • TCC Channel 1
    • Pin 11 of EXT2 - PWMH ( PD08)
    • Pin 13 of EXT2 - PWML ( PB17)
  • TCC Channel 2
    • Pin 7 of EXT2 - PWMH ( PB14)
    • Pin 14 of EXT3 - PWML ( PC22)

    • Pin 3 of EXT2 - PWMH ( PA10)
    • Pin 7 of EXT2 - PWML ( PB12)
