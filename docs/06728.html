Library Interface
a )Initialization Function
 
Name 
Description 
 
USARTx_Initialize 
Initializes given instance of the USART peripheral. 
b) Setup Functions
 
Name 
Description 
 
USARTx_SerialSetup 
Sets the USART serial communication settings dynamically. 
 
USARTx_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given USART's read events occur. 
 
USARTx_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given USART's write events occur. 
c) Transaction Functions
 
Name 
Description 
 
USARTx_Read 
Submits a read buffer to the given USART peripheral to process. 
 
USARTx_Write 
Submits a write buffer to the given USART peripheral to transfer. 
 
USARTx_ReadByte 
Submits request to read a byte of data to the given USART peripheral. 
 
USARTx_WriteByte 
Submits a byte of data to the given USART peripheral to transfer. 
d) Status Functions
 
Name 
Description 
 
USARTx_ReadIsBusy 
Returns the read request status associated with the given USART peripheral instance. 
 
USARTx_WriteIsBusy 
Returns the write request status associated with the given USART peripheral instance. 
 
USARTx_TransmitterIsReady 
Returns the hardware status of the USART Transmitter. 
 
USARTx_ReceiverIsReady 
Returns the hardware status of the USART Receiver. 
 
USARTx_ErrorGet 
Gets the error of the given USART peripheral instance. 
 
USARTx_ReadCountGet 
Returns the count of number of bytes processed for a given USART read operation. 
 
USARTx_WriteCountGet 
Returns the count of number of bytes processed for a given USART write operation. 
e) Data Types and Constants
 
Name 
Description 
 
USARTx_CALLBACK 
Defines the data type and function signature for the USART peripheral callback function. 
 
USARTx_DATA_BIT 
Defines the data type for the USART data bit length. 
 
USARTx_ERROR 
Defines the data type for the USART peripheral errors. 
 
USARTx_PARITY 
Defines the type of parity for USART peripheral. 
 
USARTx_SERIAL_SETUP 
Defines the data type for the USART serial setup. 
 
USARTx_STOP_BIT 
Defines the length of stop bits.
