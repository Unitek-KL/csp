MCAN_OBJ Structure

MCAN PLib Instance Object 

This data structure defines the MCAN PLib Instance Object.

C
typedef struct {
  uint32_t txBufferIndex;
  uint32_t * rxId;
  uint8_t * rxBuffer;
  uint8_t * rxsize;
  MCAN_STATE state;
  MCAN_CALLBACK callback;
  uintptr_t context;
  MCAN_MSG_RAM_CONFIG msgRAMConfig;
} MCAN_OBJ;
