Running The Application

 

MCU Devices 

  1. Open the Terminal application (Ex.:Tera term) on the computer for each board
  2. Connect to the DEBUG UART Virtual COM port and configure the serial settings for each board as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build and Program the application on both boards using the MPLAB X IDE.
  4. Execute the steps and observe the output as described in the following table
Project Name 
Output 
sam_e70_xult.X 

  • Press '2' to receive message from CAN Bus on terminal window of SAM E70 XULT board number 2.
  • Press '1' to transmit message to CAN Bus on terminal window of SAM E70 XULT board number 1.
  • Transmitting Message:Success will be displayed in the terminal window of SAM E70 XULT board number 1. CAN message will be displayed in the terminal window of SAM E70 XULT board number 2.

 

sam_v71_xult.X 

  • Press '2' to receive message from CAN Bus on terminal window of SAM V71 XULT board number 2.
  • Press '1' to transmit message to CAN Bus on terminal window of SAM V71 XULT board number 1.
  • Transmitting Message:Success will be displayed in the terminal window of SAM V71 XULT board number 1. CAN message will be displayed in the terminal window of SAM V71 XULT board number 2.

 

sam_rh71_ek.X 

  • Press '2' to receive message from CAN Bus on terminal window of SAM RH71 EK board number 2.
  • Press '1' to transmit message to CAN Bus on terminal window of SAM RH71 EK board number 1.
  • Transmitting Message:Success will be displayed in the terminal window of SAM RH71 EK board number 1. CAN message will be displayed in the terminal window of SAM RH71 EK board number 2.

MPU Devices

  1. Open the Terminal application (Ex.:Tera term) on the computer
  2. Connect to the DEBUG UART Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build the IAR or MPLAB X project using IAR Embedded Workbench or MPLAB X IDE respectively
  4. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  5. Execute the steps and observe the output as described in the following table
Project Name 
Output 
sam_a5d2_xult.IAR OR sam_a5d2_xult_xc32.X 

  • Send CAN messages to the SAMA5D2-XULT board from the CAN analyzer software
  • CAN message will be displayed in the terminal window
  • RGB LED will toggle in green color for every message received
  • Received message will be echoed back to the CAN analyzer software and terminal window.
