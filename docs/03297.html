NVM_WordWrite Function

Writes one word (4 bytes) at the given address.

C
uint8_t NVM_WordWrite(
    uint32_t * data, 
    const uint32_t address
);
Preconditions

None

Parameters
Parameters 
Description 
data 
Pointer to the buffer which holds data to be written.
 
address 
Address at which the data will be written. 
Returns

None.

Example
NVM_WordWrite(data_to_fill, 0x9D100000);
