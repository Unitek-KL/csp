Microchip Support
Library Interface
a) Initialization Functions
 
Name 
Description 
 
RTC_Initialize 
Initializes given instance of the RTC peripheral. 
b) Setup Functions
 
Name 
Description 
 
RTC_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the Timeout events occur. 
c) Transaction Functions
 
Name 
Description 
 
RTC_TimeGet 
Reads the current time 
 
RTC_TimeSet 
Sets the Time for the RTC peripheral. 
 
RTC_AlarmSet 
Sets up the Alarm 
 
RTC_InterruptEnable 
Enables the specified RTC interrupt 
 
RTC_InterruptDisable 
Disables the specified RTC interrupt 
d) Data Types and Constants
 
Name 
Description 
 
RTC_CALLBACK 
Defines the data type and function signature for the RTC peripheral callback function. 
 
RTC_ALARM_MASK 
Defines the data type for the RTC Alarm Mask. 
 
RTC_INT_MASK 
Defines the Interrupt mask for RTC events. 
 
RTC_INT_MASK 
Mask for enabling and disabling the RTCC interrupt.
