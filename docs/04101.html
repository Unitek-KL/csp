Library Interface
a) Initialization Function
 
Name 
Description 
 
PM_Initialize 
Initializes the PM. 
b) Transaction Functions
 
Name 
Description 
 
PM_BackupModeEnter 
Puts the device in backup mode. 
 
PM_HibernateModeEnter 
Puts the device in hibernate mode. 
 
PM_IdleModeEnter 
Puts the device in Idle mode. 
 
PM_IO_RetentionClear 
Turns off I/O retention. To be called after exiting HIBERNATE or BACKUP mode. 
 
PM_IO_RetentionSet 
Turns on I/O retention when exiting HIBERNATE or BACKUP mode. 
 
PM_OffModeEnter 
Puts the device in off mode. 
 
PM_StandbyModeEnter 
Puts the device in standby mode.
