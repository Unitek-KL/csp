Running The Application

MCU Devices

  1. Open the Terminal application (Ex.:Tera Term) on the computer.
  2. Configure the serial port settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build and program the application using the MPLAB X IDE.
  4. See the following message in the console.

  1. Type 10 characters in the terminal. It will echo back the received bytes and toggles an LED. The following table provides the LED name.

 

The following table provides the LED name:

Kit Name 
LED Name 
SAM G55 Xplained Pro Evaluation Kit 
LED 0 
SAM RH71 Evaluation Kit 
LED 0 

 

MPU Devices

  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the USB-UART Click board's Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build the IAR or MPLAB X project using IAR Embedded Workbench or MPLAB X IDE respectively
  4. Use the output binary file (harmony.bin) to setup the board as detailed in the hardware setup
  5. Once COM port connection is established, type 10 characters or more in the Tera Term Window.
  6. Observe the output as described in the following table
Project Name 
Output 
sam_a5d2_xult.IAR OR sam_a5d2_xult_xc32.X 

  • Characters are echoed back on entering the 10th character
  • RGB LED will toggle in green color for every line of character echoed

 

sam_9x60_ek.IAR 

  • Characters are echoed back on entering the 10th character
  • RGB LED will toggle in green color for every line of character echoed
