Introduction

The TC channel in capture mode is used to measure pulse width, period, timestamp. A capture trigger can be provided by input event line TC_EV or by asynchronous IO pin WO[x] for each capture channel or by a TC event. This provides below features: 

 

  • Selectable 8/16/32 bit counter
  • Input capture
    • Event / IO pin edge capture
    • Frequency capture
    • Pulse-width capture
    • Time-stamp capture
    • Minimum and maximum capture
  • Interrupts/output events on:
    • Capture
  • DMA support
