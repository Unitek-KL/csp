Running The Application

MCU Devices 

  1. Build and program the application using the MPLAB X IDE.
  2. Disconnect and reconnect the Debug USB port to power cycle the board
  3. Press the "RESET" switch and observe the LED toggling

The following table provides the LED name.

  1. Note that this demo reconfigured the external reset pin as a GPIO pin. The board must be power cycled to run the code when a new project is programmed.

 

 

MPU Devices 

  1. Build the IAR or MPLAB X project using IAR Embedded Workbench or MPLAB X IDE respectively
  2. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  3. Reset the board to run the application
  4. Press the "RESET" switch and observe the LED toggling
    • NOTE: Push and Release RESET switch very quickly so as to not end up cutting power to the PMIC triggering a full power on reset (instead of just toggling of the reset pin)
