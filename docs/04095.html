PWM_CHANNEL_NUM Type

PWM channel number 

This enumeration identifies PWM channel number in a PWM peripheral. This can be used as an argument in the APIs

C
typedef enum PWM_CHANNEL_NUM@2 PWM_CHANNEL_NUM;
Remarks

None.
