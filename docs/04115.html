SDHC_DataTransferFlags Structure

SDHC data transfer flags 

The client indicates whether the transfer contains data stage and if yes, it further specifies the data transfer direction and data transfer type. The object of this data type is passed as an argument to the SDHCx_CommandSend function.

C
typedef struct {
  bool isDataPresent;
  SDHC_DATA_TRANSFER_DIR transferDir;
  SDHC_DATA_TRANSFER_TYPE transferType;
} SDHC_DataTransferFlags;
Members
Members 
Description 
bool isDataPresent; 
Indicates if SDHC transfer has a data stage or not 
SDHC_DATA_TRANSFER_DIR transferDir; 
Direction of the data transfer 
SDHC_DATA_TRANSFER_TYPE transferType; 
Type of the data transfer - read/write 
Remarks

None.
