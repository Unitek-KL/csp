Running The Application

IAR Embedded work bench

  1. Open the Terminal application (Ex.:Tera term) on the computer
  2. Connect to the virtual COM port associated with USB_UART click board and configure the serial settings as follows:
    • Baud : 9600 bps (NOTE: RXLP does not support higher baud rates)
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build the project using IAR Embedded Workbench
  4. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup

SAMA5D2 Xplained Ultra evaluation kit 

  • RGB_LED toggles in Blue color indicating the board is up and running
  • Press switch PB_USER to shutdown the device (LED toggling stops)
  • Send a character on the USB UART Virtual COM port to reset the device
  • Device resets (indicated by the RGB_LED toggling in blue color)
