Library Interface
a) Initialization Function
 
Name 
Description 
 
CANx_Initialize 
Initializes given instance of the CAN peripheral. 
b) Setup Functions
 
Name 
Description 
 
CANx_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given CAN's transfer events occur. 
c) Transaction Functions
 
Name 
Description 
 
CANx_ErrorGet 
Returns the error during transfer. 
 
CANx_InterruptClear 
Clears Interrupt status. 
 
CANx_InterruptGet 
Returns the Interrupt status. 
 
CANx_MessageRAMConfigSet 
Set the Message RAM Configuration. 
 
CANx_MessageReceive 
Receives a message from CAN bus. 
 
CANx_MessageTransmit 
Transmits a message into CAN bus. 
d) Status Functions
 
Name 
Description 
 
CANx_IsBusy 
Returns the Peripheral busy status. 
e) Data Types and Constants
 
Name 
Description 
 
CAN_CALLBACK 
CAN Callback Function Pointer. 
 
CAN_ERROR 
CAN Transfer Error data type. 
 
CAN_INTERRUPT_MASK 
CAN Interrupt Mask. 
 
CAN_MODE 
CAN Mode for Classic CAN and CAN FD. 
 
CAN_MSG_RAM_CONFIG 
CAN Message RAM Configuration structure. 
 
CAN_MSG_RX_ATTRIBUTE 
CAN Rx Message Attribute for Rx Buffer/FIFO0/FIFO1. 
 
CAN_MSG_TX_ATTRIBUTE 
CAN Tx Message Attribute for Tx Buffer/FIFO. 
 
CAN_OBJ 
CAN PLib Object structure. 
 
CAN_STATE 
CAN PLib Task State.
