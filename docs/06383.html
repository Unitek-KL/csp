Library Interface

This section describes the Application Programming Interface (API) functions of the Peripheral Library. 

Refer to each section for a detailed description.

a) Initialization Function
 
Name 
Description 
 
TCCx_PWMInitialize 
Initializes given instance of TCC 
b) Setup Functions
 
Name 
Description 
 
TCCx_PWMCallbackRegister 
Registers the function to be called from interrupt. 
c) Transaction Functions
 
Name 
Description 
 
TCCx_PWMStart 
Starts PWM generation of given TCC instance 
 
TCCx_PWMStop 
Stops PWM generation of given TCC instance 
 
TCCx_PWMDeadTimeSet 
Sets the dead time of a given TCC module. 
 
TCCx_PWMForceUpdate 
Updates double buffer register value to actual register asynchronously 
 
TCCx_PWM16bitCounterSet 
Sets the counter value of a given TCC instance 
 
TCCx_PWM16bitDutySet 
Sets the duty value of a given TCC channel. 
 
TCCx_PWM16bitPeriodGet 
Reads the period value of a given TCC instance. 
 
TCCx_PWM16bitPeriodSet 
Sets the period value of a given TCC instance. 
 
TCCx_PWM24bitCounterSet 
Sets the counter value of a given TCC instance 
 
TCCx_PWM24bitDutySet 
Sets the duty value of a given TCC channel. 
 
TCCx_PWM24bitPeriodGet 
Reads the period value of a given TCC instance. 
 
TCCx_PWM24bitPeriodSet 
Sets the period value of a given TCC instance. 
 
TCCx_PWMPeriodInterruptDisable 
Disables period interrupt 
 
TCCx_PWMPeriodInterruptEnable 
Enables period interrupt 
d) Data Types and Constants
 
Name 
Description 
 
TCC_CALLBACK 
Defines the function pointer data type and function signature for the tcc callback function. 
 
TCC_CALLBACK_OBJECT 
Callback structure 
 
TCCx_CHANNEL_NUM 
Identifies channel number within TCC module 
d) Status Functions
 
Name 
Description 
 
TCCx_PWMInterruptStatusGet 
Reads the interrupt flags
