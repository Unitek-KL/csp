MCPWM_ChannelPinsOverrideDisable Function

This function disables override function for PWM output pins PWMH and PWML. When override is disabled, PWM generation logic controls the PWMH and PWML states.

C
void MCPWM_ChannelPinsOverrideDisable(
    MCPWM_CH_NUM channel
);
Preconditions

MCPWM_Initialize() must have been called first for the associated instance.

Parameters
Parameters 
Description 
channel 
MCPWM channel number 
Returns

None.

Remarks

None

Example
    MCPWM_Initialize();
    MCPWM_ChannelPinsOverrideDisable(MCPWM_CH_0);
