Library Interface
a) Initialization Function
 
Name 
Description 
 
FLEXCOMx_USART_Initialize 
Initializes given instance of the FLEXCOM USART peripheral. 
b) Setup Functions
 
Name 
Description 
 
FLEXCOMx_USART_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given FLEXCOM USART's read events occur. 
 
FLEXCOMx_USART_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given FLEXCOM USART's write events occur. 
c) Transaction Functions
 
Name 
Description 
 
FLEXCOMx_USART_Read 
Submits a read buffer to the given FLEXCOM USART peripheral to process. 
 
FLEXCOMx_USART_ReadByte 
Submits request to read a byte of data to the given FLEXCOM USART peripheral. 
 
FLEXCOMx_USART_ReadCountGet 
Returns the count of number of bytes processed for a given FLEXCOM USART read operation. 
 
FLEXCOMx_USART_SerialSetup 
Sets the FLEXCOM USART serial communication settings dynamically. 
 
FLEXCOMx_USART_Write 
Submits a write buffer to the given FLEXCOM USART peripheral to transfer. 
 
FLEXCOMx_USART_WriteByte 
Submits a byte of data to the given FLEXCOM USART peripheral to transfer. 
 
FLEXCOMx_USART_WriteCountGet 
Returns the count of number of bytes processed for a given FLEXCOM USART write operation. 
d) Status Functions
 
Name 
Description 
 
FLEXCOMx_USART_ErrorGet 
Gets the error of the given FLEXCOM USART peripheral instance. 
 
FLEXCOMx_USART_ReadIsBusy 
Returns the read request status associated with the given FLEXCOM USART peripheral instance. 
 
FLEXCOMx_USART_WriteIsBusy 
Returns the write request status associated with the given FLEXCOM USART peripheral instance. 
 
FLEXCOMx_USART_ReceiverIsReady 
Returns the hardware status of the FLEXCOM USART Receiver. 
 
FLEXCOMx_USART_TransmitterIsReady 
Returns the hardware status of the FLEXCOM USART Transmitter. 
e) Data Types and Constants
 
Name 
Description 
 
FLEXCOM_USART_CALLBACK 
Defines the data type and function signature for the FLEXCOM USART peripheral callback function. 
 
FLEXCOM_USART_DATA 
Defines the data type for the FLEXCOM USART serial setup. 
 
FLEXCOM_USART_ERROR 
Defines the data type for the FLEXCOM USART peripheral errors. 
 
FLEXCOM_USART_PARITY 
This is type FLEXCOM_USART_PARITY. 
 
FLEXCOM_USART_SERIAL_SETUP 
This is type FLEXCOM_USART_SERIAL_SETUP. 
 
FLEXCOM_USART_STOP 
This is type FLEXCOM_USART_STOP.
