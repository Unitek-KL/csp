TCx_Compare32bitCounterSet Function

This function sets new timer counter value.

C
void TCx_Compare32bitCounterSet(
    uint32_t count
);
Preconditions

TCx_CompareInitialize() function must have been called first.

Parameters
Parameters 
Description 
count 
new counter value to set. 
Returns

None.

Remarks

The caller must know the number of significant bytes of timer. Counter value is right-aligned.

Example
  uint32_t count = 0x100;
  TCx_CompareInitialize();
  TCx_CompareStart();
  TCx_Compare32bitCounterSet(count);
