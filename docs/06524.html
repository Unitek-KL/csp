Library Interface
a) Initialization Function
 
Name 
Description 
 
XDMAC_Initialize 
Initializes the XDMAC controller of the device. 
b) Setup Functions
 
Name 
Description 
 
XDMAC_ChannelCallbackRegister 
This function allows a XDMAC PLIB client to set an event handler. 
 
XDMAC_ChannelSettingsGet 
Returns the channel settings of a specific XDMAC Channel. 
 
XDMAC_ChannelSettingsSet 
Sets the channel settings of a specific XDMAC Channel. 
c) Transaction Functions
 
Name 
Description 
 
XDMAC_ChannelTransfer 
Adds a data transfer to a XDMAC channel and enables the channel to start data transfer. 
 
XDMAC_ChannelLinkedListTransfer 
Sets up a multi-block data transfer on a specified XDMAC channel using linked list feature. 
 
XDMAC_ChannelDisable 
Disables the specified channel. 
 
XDMAC_ChannelBlockLengthSet 
Sets the channel Block Length of a specific XDMAC Channel. 
d) Status Functions
 
Name 
Description 
 
XDMAC_ChannelIsBusy 
Returns the busy status of a specific XDMAC Channel. 
e) Data Types and Constants
 
Name 
Description 
 
XDMAC_CHANNEL_CALLBACK 
Pointer to a XDMAC Transfer Event handler function. 
 
XDMAC_CHANNEL 
This lists the set of channels available for data transfer using XDMAC. 
 
XDMAC_CHANNEL_CONFIG 
Defines the channel settings. 
 
XDMAC_DESCRIPTOR_CONTROL 
Defines the descriptor control for linked list operation. 
 
XDMAC_DESCRIPTOR_VIEW_0 
View 0 
 
XDMAC_DESCRIPTOR_VIEW_1 
View 1 
 
XDMAC_DESCRIPTOR_VIEW_2 
View 2 
 
XDMAC_DESCRIPTOR_VIEW_3 
View 3 
 
XDMAC_MICRO_BLOCK_CONTROL 
Defines the control parameters for linked list operation. 
 
XDMAC_TRANSFER_EVENT 
Enumeration of possible XDMAC transfer events.
