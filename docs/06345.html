UART_SERIAL_SETUP Structure

UART Serial Setup 

This can be used to define a serial setup which may then be used to change the serial setup of the UART dynamically.

C
typedef struct {
  uint32_t baudRate;
  UART_PARITY parity;
} UART_SERIAL_SETUP;
Remarks

None.
