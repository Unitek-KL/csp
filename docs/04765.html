Running The Application

Running project with MPLAB X IDE 

  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. SAM E70 Xplained Ultra Evaluation Kit and SAM V71 Xplained Ultra Evaluation Kit
    • Connect PA00 (Pin 7 of EXT1) to PC30 (Pin 8 of EXT1)
  4. SAM C21n Xplained Pro Evaluation Kit
    • Connect PA09 (Pin 4 of EXT2) to PA21 (Pin 6 of EXT2)
  5. SAM D21 Xplained Pro Evaluation Kit and SAM D20 Xplained Pro Evaluation Kit
    • Connect PA19 (Pin 18 of EXT2) to PA16 (pin 17 of EXT2)
  6. SAM E54 Xplained Pro Evaluation Kit
    • Connect PA05 (Pin 13 of EXT1) to PB04 (Pin 3 of EXT1)
  7. Build and Program the application using the MPLAB X IDE.
  8. Console displays the frequency and duty cycle of the input signal.
  9. Observe the output as described in the following table:
Project Name 
Output 
sam_c21n_xpro.X 
Success: Console displays frequency and duty cycle of input signal. Frequency is constant (800 Hz) and duty cycle changes by 1%.

sam_v71_xult.X 
Success: Console displays frequency and duty cycle of input signal. Frequency is constant (2500 Hz) and duty cycle changes by 1%.
 

 

Running project with IAR Embedded work bench

  1. Build the project using IAR Embedded Workbench
  2. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  3. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  4. Reset the board to run the application
