HSMCI_DataErrorGet Function

Returns the errors associated with a data transfer. For a data transfer, the returned value is a bit wise OR of the following errors, defined in the HSMCI_ERROR_FLAGS enum. HSMCI_DATA_TIMEOUT_ERROR = 0x0010, HSMCI_DATA_CRC_ERROR = 0x0020, HSMCI_DATA_UNDERRUN_ERROR = 0x0040, HSMCI_DATA_OVERRUN_ERROR = 0x0080 The error flags are cleared after a call to this API.

C
uint16_t HSMCI_DataErrorGet();
Preconditions

A command containing a data stage must have been initiated using the HSMCI_CommandSend() function.

Returns

Bit wise OR of the data error flags defined under the enum HSMCI_ERROR_FLAGS - HSMCI_DATA_TIMEOUT_ERROR, HSMCI_DATA_CRC_ERROR, HSMCI_DATA_UNDERRUN_ERROR, HSMCI_DATA_OVERRUN_ERROR.

Remarks

None.

Example
uint16_t data_error = HSMCI_DataErrorGet();

if (data_error & (HSMCI_DATA_TIMEOUT_ERROR | HSMCI_DATA_CRC_ERROR |
            HSMCI_DATA_UNDERRUN_ERROR | HSMCI_DATA_OVERRUN_ERROR))
{
    // Handle data error
}
