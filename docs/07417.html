TCx_CAPTURE_CALLBACK Type

Callback Function Pointer for Capture mode 

This data type defines the function pointer and function signature for the tc channel callback function. The library will call back the client's function with this signature from the interrupt routine.

C
typedef void (* TCx_CAPTURE_CALLBACK)(TCx_CAPTURE_STATUS status, uintptr_t context);
Preconditions

TCx_CHy_Initialize must have been called for the given TC channel instance and TCx_CHy_CaptureCallbackRegister must have been called to register the function to be called.

Parameters
Parameters 
Description 
status 
Interrupt status in capture mode 
context 
Allows the caller to provide a context value (usually a pointer to the callers context for multi-instance clients). 
Returns

None.

Remarks

None.

Example
void TC_CallbackFn (TCx_CAPTURE_STATUS status, uintptr_t context );

TC0_CH1_CaptureCallbackRegister(TC_CallbackFn, NULL);
