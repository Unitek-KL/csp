UARTx_SerialSetup Function

This function sets the UART serial communication settings dynamically.

C
bool UARTx_SerialSetup(
    UART_SERIAL_SETUP * setup, 
    uint32_t srcClkFreq
);
Preconditions

UARTx_Initialize must have been called for the associated UART instance. 

The UART transmit or receive transfer status should not be busy.

Parameters
Parameters 
Description 
setup 
Pointer to the structure containing the serial setup. 
srcClkFreq 
UART Peripheral Clock Source Frequency. 
Returns
  • true - Serial setup was updated Successfully.
  • false - Failure while updating serial setup.
Remarks

srcClkFreq overrides any change in the peripheral clock frequency. If configured to zero, PLIB takes the peripheral clock frequency from MHC.

Example
UART_SERIAL_SETUP setup = {
        115200,
        UART_PARITY_ODD
    };

UART1_SerialSetup(&setup, 0);
