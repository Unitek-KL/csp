Hardware Setup
  1. Project sam_c21n_xpro.X
    • Hardware Used
      • SAM C21N Xplained Pro Evaluation Kit
    • Hardware Setup
      • Connect mikroBUS Xplained Pro to the EXT2 header of the SAM C21N Xplained Pro Evaluation Kit
      • Plug the EEPROM 4 Click Board into the MikroBus socket of the mikroBUS Xplained Pro
      • Connect the Debug USB port on the board to the computer using a micro USB cable
