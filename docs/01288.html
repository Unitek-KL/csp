Running The Application
  1. Connect a logic analyzer to SCK pin.
  2. Connect a logic analyzer to MOSI pin.
  3. Connect a logic analyzer to the Manchester-encoded output CCL_OUT pin.
  4. Build and Program the application using the MPLAB X IDE.
