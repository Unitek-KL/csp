Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
ADCHSx_Initialize 
Initializes given instance of ADCHS peripheral. 
b) Setup Functions
 
Name 
Description 
 
ADCHSx_CallbackRegister 
Registers the function to be called from interrupt 
c) Transaction Functions
 
Name 
Description 
 
ADCHSx_ChannelConversionStart 
Starts the ADC conversion of an analog channel 
 
ADCHSx_ChannelEarlyInterruptDisable 
Disables the ADC channel early result data interrupt 
 
ADCHSx_ChannelEarlyInterruptEnable 
Enables the ADC channel early result data interrupt 
 
ADCHSx_ChannelResultGet 
Reads the conversion result of the channel 
 
ADCHSx_ChannelResultInterruptDisable 
Disables the ADC channel result data interrupt 
 
ADCHSx_ChannelResultInterruptEnable 
Enables the ADC channel result data interrupt 
 
ADCHSx_GlobalEdgeConversionStart 
Starts the ADC conversion of analog channels configured for global software edge trigger 
 
ADCHSx_GlobalLevelConversionStart 
Starts the ADC conversion of analog channels configured for global software level trigger 
 
ADCHSx_ModulesDisable 
Disables the ADC modules 
 
ADCHSx_ModulesEnable 
Enables the ADC modules 
d) Status Functions
 
Name 
Description 
 
ADCHSx_ChannelResultIsReady 
Returns the status of the channel conversion 
e) Data Types and Constants
 
Name 
Description 
 
ADCHS_CALLBACK 
Defines the function pointer data type and function signature for the adchs peripheral callback function. 
 
ADCHS_CALLBACK_OBJECT 
Callback structure 
 
ADCHS_CHANNEL_NUM 
Identifies ADCHS inputs as analog channel number 
 
ADCHS_MODULE_MASK 
Identifies dedicated and shared ADC module mask
