Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
QEIx_Initialize 
Initializes given instance of QEI peripheral in quadrature mode. 
b) Setup Functions
 
Name 
Description 
 
QEIx_CallbackRegister 
Registers the function to be called from interrupt. 
c) Transaction Functions
 
Name 
Description 
 
QEIx_Start 
Starts the given QEI instance counter. 
 
QEIx_Stop 
Stops the given QEI instance counter. 
 
QEIx_PositionGet 
Reads the position from the quadrature encoder. 
 
QEIx_RevolutionsGet 
Reads the number of revolutions tracked by the quadrature encoder. 
 
QEIx_VelocityGet 
Reads the velocity counter from the quadrature encoder. 
d) Data Types and Constants
 
Name 
Description 
 
QEI_CALLBACK 
Defines the function pointer data type and function signature for the qei peripheral callback function. 
 
QEI_CALLBACK_OBJ 
Callback structure 
 
QEI_INTERRUPT 
Identifies Quadrature mode interrupt source . 
 
QEI_STATUS 
Identifies Quadrature mode interrupt source mask.
