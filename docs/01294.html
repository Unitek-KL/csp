dac_wav_gen_dma

The DAC Peripheral library is used in hardware trigger mode to generate a Sine wave. The TC peripheral is configured to generate a trigger every two microseconds. Trigger is connected to the DAC using the EVSYS peripheral. 

The DMA is used to setup a linked list to transfer the sine wave samples from the lookup table to the DAC "DATABUF" register. In this application the number of DAC samples per output wave (sine) is 100.
