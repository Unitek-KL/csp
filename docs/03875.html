Configuring the Library

Configure the peripheral library using the MHC. 

 

Voltage Regulator Operation in Standby Sleep Mode: 

There are also two different operating modes for the internal regulator viz., Normal mode, and Low Power mode. In standby mode, the power manager selects either the main or the low power voltage regulator to supply the VDDCORE.

  • Low Power: By default the low power voltage regulator is used to minimize the power consumption in standby mode.
  • Performance Oriented: If a sleepwalking task is working on either asynchronous clocks (generic clocks) or synchronous clock (APB/AHB clocks), the main voltage regulator is used.
  • Automatic Mode: The regulator operates in low-power mode (LP VREG) by default and can switch automatically to the main regulator if a task required by a peripheral requires more power. It returns automatically in the low power mode as soon as the task is completed

 

RAM State in Standby Sleep mode. 

In standby sleep mode, RAM is put in low-power mode by default. This means that the RAM is back-biased, and the DMAC cannot access it. This behavior can be changed by configuring the Back Bias bit in the Standby Configuration register (STDBYCFG.BBIASHS).
