Running The Application
  1. Open the QSPI XIP image project (xip_image/firmware/sam_e70_xult.X) in the MPLAB X IDE
  2. Build the project using the MPLAB X IDE and do not program.
  3. Run python script (xip_image_pattern_gen.py) to extract the hex code from the binary and store it in the header file (xip_image_pattern_hex.h) as an array of hex values.
   python xip_image_pattern_gen.py
  1. Open the QSPI main project (xip_main/firmware/ sam_e70_xult.X) in the MPLAB X IDE
  2. Build and program the application using the MPLAB X IDE.
  3. The main application programs the QSPI XIP image and executes the code from serial Flash memory. The QSPI XIP image application blinks the LED continuously.
