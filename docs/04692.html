Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
SYSTICK_TimerInitialize 
Initializes SysTick. 
b) Setup Functions
 
Name 
Description 
 
SYSTICK_TimerCallbackSet 
Sets the pointer to the function (and it's context) to be called when the given systick reaches 0. 
c) Transaction Functions
 
Name 
Description 
 
SYSTICK_TimerCounterGet 
Get the SysTick current Value 
 
SYSTICK_TimerFrequencyGet 
Get the SysTick frequency Value 
 
SYSTICK_TimerPeriodGet 
Get the SysTick Load Value 
 
SYSTICK_TimerStart 
Starts System Timer. 
 
SYSTICK_TimerStop 
Stops System Timer. 
 
SYSTICK_TimerPeriodSet 
Set the SysTick Load Value 
 
SYSTICK_TimerRestart 
Restarts SysTick. 
 
SYSTICK_DelayMs 
Blocking function to generate delay in milliseconds 
d) Status Functions
 
Name 
Description 
 
SYSTICK_TimerPeriodHasExpired 
Returns the current status of the systick 
e) Data Types and Constants
 
Name 
Description 
 
SYSTICK_CALLBACK 
Defines the data type and function signature for the SysTick callback function. 
 
SYSTICK_OBJECT 
Defines the data type for the SYSTICK OBJECT.
