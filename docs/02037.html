EFC_ERROR Enumeration

This enum is used to define error encountered during last Flash operation.

C
typedef enum {
  EFC_ERROR_NONE = 0x1,
  EFC_CMD_ERROR = 0x2,
  EFC_LOCK_ERROR = 0x4,
  EFC_FLERR_ERROR = 0x8,
  EFC_ECC_ERROR = 0xF0000
} EFC_ERROR;
Members
Members 
Description 
EFC_ERROR_NONE = 0x1 
No Error Occured 
EFC_CMD_ERROR = 0x2 
In-valid command 
EFC_LOCK_ERROR = 0x4 
Flash region is locked 
EFC_FLERR_ERROR = 0x8 
Flash Error 
EFC_ECC_ERROR = 0xF0000 
Flash Encountered an ECC error 
Remarks

None.
