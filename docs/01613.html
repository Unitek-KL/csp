EVSYSx_InterruptDisable Function

This function disables the specified evsys interrupt.

C
void EVSYSx_InterruptDisable(
    EVSYS_CHANNEL channel, 
    EVSYSx_INT_MASK interrupt
);
Preconditions

EVSYSx module should be initialized with the required configuration parameters from the MHC GUI in the EVSYSx_Initialize() function

Parameters
Parameters 
Description 
channel 
Channel for which the interrupt needs to be disabled 
interrupt 
Interrupt to be Disabled 
Returns

None

Remarks

Interrupts configuration will be generated in EVSYSx_Initialize() function as per user settings in MHC. The EVSYSx_InterruptEnable and EVSYSx_InterruptDisable API should be used to enable and disable interrupt at run time.

Example
    // Disable Event System channel 0 Event Detection interrupt 
    EVSYSx_InterruptDisable(EVSYS_CHANNEL_0, EVSYS_INT_EVD);
