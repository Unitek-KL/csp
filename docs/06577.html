Library Interface
a) Initialization Function
 
Name 
Description 
 
SERCOMx_I2C_Initialize 
Initializes the instance of the SERCOM peripheral operating in I2C mode. 
b) Setup Functions
 
Name 
Description 
 
SERCOMx_I2C_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given SERCOM I2C's transfer events occur. 
 
SERCOMx_I2C_TransferSetup 
Dynamic setup of TWIHS Peripheral. 
c) Transaction Functions
 
Name 
Description 
 
SERCOMx_I2C_Read 
Reads data from the slave. 
 
SERCOMx_I2C_Write 
Writes data to the slave. 
 
SERCOMx_I2C_WriteRead 
Write and Read data from Slave. 
d) Status Functions
 
Name 
Description 
 
SERCOMx_I2C_ErrorGet 
Returns the latest error that occurred on the bus. 
 
SERCOMx_I2C_IsBusy 
Returns the Peripheral busy status. 
e) Data Types and Constants
 
Name 
Description 
 
SERCOM_I2C_CALLBACK 
Defines the data type and function signature for the SERCOM I2C peripheral callback function. 
 
SERCOM_I2C_ERROR 
Defines the possible errors that the SERCOM I2C peripheral can generate. 
 
SERCOM_I2C_TRANSFER_SETUP 
I2C transfer setup data structure
