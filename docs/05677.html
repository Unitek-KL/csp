SPIx_TRANSFER_SETUP Structure

SPI Setup Parameters 

This structure identifies the possible setup parameters for SPI which can be changed dynamically if needed.

C
typedef struct {
  uint32_t clockFrequency;
  SPIx_CLOCK_PHASE clockPhase;
  SPIx_CLOCK_POLARITY clockPolarity;
  SPIx_DATA_BITS dataBits;
} SPIx_TRANSFER_SETUP;
Members
Members 
Description 
uint32_t clockFrequency; 
Baud Rate or clock frequency 
SPIx_CLOCK_PHASE clockPhase; 
Clock Phase 
SPIx_CLOCK_POLARITY clockPolarity; 
Clock Polarity 
SPIx_DATA_BITS dataBits; 
Number of bits per transfer 
Remarks

None.
