Running The Application

Running project with MPLAB X IDE 

  1. Build and Program the application using the MPLAB X IDE.
  2. Observe the output as described in the following table:

 

Project Name 
Output 
sam_c21n_xpro.X 
Success: Observe waveforms on oscilloscope.

  • Observe PB31 "Pin 8 of EXT2 (TC0_WO1)" on the oscilloscope which generates an active low output with frequency 75 Hz
  • Observe PA15 "Pin 6 (D5) of J804 (TC3_WO1)" on the oscilloscope which generates an active high output with frequency 75 Hz
  • Observe PB12 "Pin 7 of EXT1 (TC4_WO0)" on the oscilloscope which generates the toggle output with frequency 37.5 Hz

 

sam_d20_xpro.X 
Success: Observe waveforms on oscilloscope.

  • Observe PA19 "Pin 18 of EXT2 (TC3_WO1)" on the oscilloscope which generates an active low output with 75 Hz
  • Observe PB09 "Pin 13 of EXT1 (TC4_WO1)" on the oscilloscope which generates an active high output with 75 Hz
  • Observe PB03 "Pin 8 of EXT1 (TC6_WO1)" on the oscilloscope which generates the toggle output with 37.5 Hz

 

sam_d21_xpro.X 
Success: Observe waveforms on oscilloscope.

  • Observe PA19 "Pin 18 of EXT2 (TC3_WO1)" on the oscilloscope which generates an active low output with 75 Hz
  • Observe PB09 "Pin 13 of EXT1 (TC4_WO1)" on the oscilloscope which generates an active high output with 75 Hz
  • Observe PB03 "Pin 8 of EXT1 (TC6_WO1)" on the oscilloscope which generates the toggle output with 37.5 Hz

 

sam_e54_xpro.X 
Success: Observe waveforms on oscilloscope.

  • Observe PA04 "Pin 14 of EXT1 (TC0_WO0)" on the oscilloscope which generates the toggle output with 37.5 Hz
  • Observe PA07 "Pin 6 of EXT1 (TC1_WO1)" on the oscilloscope which generates an active high output with 75 Hz
  • Observe PB09 "Pin 8 of EXT1 (TC4_WO1)" on the oscilloscope which generates an active low output with 75 Hz

Running project with IAR Embedded work bench 

  1. Build the project using IAR Embedded Workbench
  2. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  3. Reset the board to run the application

 

Project Name 
Output 
sam_a5d2_xult.IAR 

  • Observe PD29 "Pin 10 of XPRO EXT1 " on the oscilloscope which generates an active high output with 1 Hz frequency
  • Observe PD30 "PIN 8 of J8" on oscilloscope which generates an active low output with 1 Hz frequency.

 

sam_9x60_ek.IAR 

  • Observer PA22 (Pin labeled TX on Mikrobus connector) on oscilloscope which generates an active high output with 100 Hz frequency and 75% duty cycle.
