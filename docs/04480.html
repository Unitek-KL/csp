USARTx_STOP Enumeration

USART STOP 

This may be used to check the type of stop bits with the USART peripheral during serial setup.

C
typedef enum {
  USART_STOP_1_BIT,
  USART_STOP_2_BIT,
  USART_STOP_INVALID
} USARTx_STOP;
Members
Members 
Description 
USART_STOP_1_BIT 
1 bit stop bit 
USART_STOP_2_BIT 
2 bit stop bit 
USART_STOP_INVALID 
Force the compiler to reserve 32-bit memory for each enum 
Remarks

None.
