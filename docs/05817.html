rswdt_timeout

This example demonstrates the usage of Reinforced Safety Watchdog Timer to avoid system deadlock. The simplest way to emulate a deadlock is to wait in an infinite loop. The application enters such an infinite loop when the switch is pressed. As a result the RSWDT counter is not cleared within a bounded time configured using MHC. The RSWDT peripheral will trigger a system reset as the counter exceeds the specified time band.
