FLEXCOM_TWI_CALLBACK Type

FLEXCOM TWI Transfer Event Callback Function Pointer. 

This data type defines the function signature for the FLEXCOM TWI peripheral callback function. The FLEXCOM TWI peripheral will call back the client's function with this signature when the FLEXCOM TWI Transfer event has occurred.

C
typedef void (* FLEXCOM_TWI_CALLBACK)(uintptr_t contextHandle);
Preconditions

FLEXCOMx_TWI_Initialize must have been called for the given FLEXCOM TWI peripheral instance and FLEXCOMx_TWI_CallbackRegister must have been called to set the function to be called.

Parameters
Parameters 
Description 
status 
Complete or error status of the FLEXCOM TWI transfer.
 
context 
Allows the caller to provide a context value (usually a pointer to the callers context for multi-instance clients). 
Returns

None.

Remarks

None.

Example
// Data tracking what each of my instances is doing.
MY_DATA myData[2];

void MyFLEXCOMTWICallback (uintptr_t context )
{
    MY_DATA *mePtr = (MY_DATA *)context;

    //Handle complete status or error here based on context 
}

FLEXCOM1_TWI_CallbackSet(MyFLEXCOMTWICallback, &myData[0]);
FLEXCOM2_TWI_CallbackSet(MyFLEXCOMTWICallback, &myData[1]);
