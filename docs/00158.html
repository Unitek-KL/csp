Introduction

The Analog-to-Digital Converter (ADC) peripheral supports following features.

  • Configurable resolution: 8-, 10- or 12-bit results.
  • Sampling rate of up to 1MSPS.
  • Differential and single-ended measurements can be performed.
  • Software or Hardware trigger to start conversion
  • Both internal and external reference voltages can be used.
  • Compare function for accurate monitoring of user-defined thresholds, with minimum software intervention required..
  • Signed and unsigned results.
  • Left- or right aligned result to ease calculation when the result is represented as a signed value.
  • Master slave mode to synchronize two ADC modules
