I2C_STATE Enumeration

I2C State. 

This data type defines the I2C PLIB Task States.

C
typedef enum {
  I2C_STATE_START_CONDITION,
  I2C_STATE_ADDR_BYTE_1_SEND,
  I2C_STATE_ADDR_BYTE_2_SEND,
  I2C_STATE_WRITE,
  I2C_STATE_READ,
  I2C_STATE_READ_BYTE,
  I2C_STATE_WAIT_ACK_COMPLETE,
  I2C_STATE_WAIT_STOP_CONDITION_COMPLETE,
  I2C_STATE_IDLE
} I2C_STATE;
Remarks

None.
