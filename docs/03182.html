MCPWM_CH_NUM Enumeration

MCPWM channel number 

This enumeration identifies MCPWM channel number in a MCPWM peripheral. This can be used as an argument in the APIs

C
typedef enum {
  MCPWM_CH_0,
  MCPWM_CH_1,
  MCPWM_CH_2,
  MCPWM_CH_3,
  MCPWM_CH_4,
  MCPWM_CH_5,
  MCPWM_CH_6,
  MCPWM_CH_7,
  MCPWM_CH_8,
  MCPWM_CH_9,
  MCPWM_CH_10,
  MCPWM_CH_11
} MCPWM_CH_NUM;
Remarks

None.
