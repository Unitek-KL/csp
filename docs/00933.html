Library Interface
a) Initialization Function
 
Name 
Description 
 
CANx_Initialize 
Initializes given instance of the CAN peripheral. 
b) Setup Functions
 
Name 
Description 
 
CANx_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given CAN's transfer events occur. 
c) Transaction Functions
 
Name 
Description 
 
CANx_MessageTransmit 
Transmits a message into CAN bus. 
 
CANx_MessageReceive 
Receives a message from CAN bus. 
 
CANx_MessageAcceptanceFilterSet 
Set Message acceptance filter configuration. 
 
CANx_MessageAcceptanceFilterGet 
Get Message acceptance filter configuration. 
 
CANx_MessageAcceptanceFilterMaskSet 
Set Message acceptance filter mask configuration. 
 
CANx_MessageAcceptanceFilterMaskGet 
Get Message acceptance filter mask configuration. 
 
CANx_MessageAbort 
Abort request for a FIFO. 
 
CANx_TransmitEventFIFOElementGet 
Get the Transmit Event FIFO Element for the transmitted message. 
d) Status Functions
 
Name 
Description 
 
CANx_ErrorGet 
Returns the error during transfer. 
 
CANx_ErrorCountGet 
Returns the transmit and receive error count during transfer. 
 
CANx_InterruptGet 
Returns the FIFO Interrupt status. 
 
CANx_IsBusy 
Returns the Peripheral busy status. 
e) Data Types and Constants
 
Name 
Description 
 
CAN_MODE 
CAN Mode for Classic CAN and CAN FD. 
 
CAN_MSG_TX_ATTRIBUTE 
CAN Tx Message Attribute for Tx FIFO and Tx Queue. 
 
CAN_FIFO_INTERRUPT_FLAG_MASK 
CAN FIFO Interrupt Status Flag Mask. 
 
CAN_ERROR 
CAN Transfer Error data type. 
 
CAN_STATE 
CAN PLib Task State. 
 
CAN_TX_MSG_OBJECT 
CAN Transmit Message Object structure. 
 
CAN_RX_MSG_OBJECT 
CAN Receive Message Object structure. 
 
CAN_TX_EVENT_FIFO_ELEMENT 
CAN Trasmit Event FIFO Element structure. 
 
CAN_RX_MSG 
CAN RX Message Buffer structure. 
 
CAN_CALLBACK 
CAN Callback Function Pointer. 
 
CAN_OBJ 
CAN PLib Object structure.
