Running The Application
  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Configure the serial port settings as follows and the following table provides serial COM port details:

  • Baud : 115200
  • Data : 8 Bits
  • Parity : None
  • Stop : 1 Bit
  • Flow Control : None
  1. Build and program the application using the MPLAB X IDE.
  2. The LED indicates the success or failure:
    • The LED is turned ON when the DMAC memory transfer successful.
