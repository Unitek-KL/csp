Introduction

The High Speed Analog to Digital Converter (ADCHS) peripheral supports the following features: 

 

  • Six ADC modules with dedicated Sample and Hold (S&H) circuits out of which five are dedicated ADC modules (ADC0 - ADC4) and one is shared ADC module (ADC7)
  • Selectable Single-ended or Differential Input Voltage
  • Input scan feature
  • Configurable trigger source for dedicated ADC modules and shared ADC module
  • Up to 16-bit Resolution by Digital Filtering
  • Early interrupt generation resulting in faster processing of converted data
