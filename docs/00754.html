AFEC Applications

This section provides help for the Analog Front-End Controller (AFEC) peripheral applications.

Topics
Name 
Description 
afec_dma_striding 
This example demonstrates taking 100 samples of three analog inputs at a 5 KHz sampling rate in triggered conversion mode, and then sorting the data by an analog channel using XDMAC. 
afec_polled_mode 
This example demonstrates how to sample an analog input in polled mode and send the converted data to console. 
afec_user_sequence 
This example demonstrates how to sample three analog inputs using the user sequencer in triggered conversion mode and send the converted data to the console.
