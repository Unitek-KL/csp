Library Interface
a) Initialization Function
 
Name 
Description 
 
DBGU_Initialize 
Initializes given instance of the DBGU peripheral. 
b) Setup Functions
 
Name 
Description 
 
DBGU_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given DBGU's read events occur. 
 
DBGU_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given DBGU's write events occur. 
c) Transaction Functions
 
Name 
Description 
 
DBGU_Read 
Submits request to read n-Bytes of data to the given DBGU peripheral. 
 
DBGU_ReadByte 
Submits request to read a byte of data to the given DBGU peripheral. 
 
DBGU_ReadCountGet 
Returns the count of number of bytes processed for a given DBGU read operation. 
 
DBGU_SerialSetup 
Sets the DBGU serial communication settings dynamically. 
 
DBGU_Write 
Submits a write buffer to the given DBGU peripheral to transfer. 
 
DBGU_WriteByte 
Submits a byte of data to the given DBGU peripheral to transfer. 
 
DBGU_WriteCountGet 
Returns the count of number of bytes processed for a given DBGU write operation. 
d) Status Functions
 
Name 
Description 
 
DBGU_ReadIsBusy 
Returns the read request status associated with the given DBGU peripheral instance. 
 
DBGU_ReceiverIsReady 
Returns the hardware status of the DBGU Receiver. 
 
DBGU_TransmitterIsReady 
Returns the hardware status of the DBGU Transmitter. 
 
DBGU_WriteIsBusy 
Returns the write request status associated with the given DBGU peripheral instance. 
 
DBGU_ErrorGet 
Gets the error of the given DBGU peripheral instance. 
e) Data Types and Constants
 
Name 
Description 
 
DBGU_CALLBACK 
Defines the data type and function signature for the DBGU peripheral callback function. 
 
DBGU_ERROR 
Defines the data type for the DBGU peripheral errors. 
 
DBGU_PARITY 
Defines the type of parity for DBGU peripheral. 
 
DBGU_SERIAL_SETUP 
Defines the data type for the DBGU serial setup.
