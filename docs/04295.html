Running The Application

Running project with IAR Embedded work bench

  1. Build the project using IAR Embedded Workbench
  2. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  3. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  4. Reset the board to run the application
  • Note: In addition to program output, bootloader outputs its log messages to the same console. These messages will be displayed immediately after reset and are not captured here.

Observe waveforms with following characteristics:

  • PWM0 generates an active high PWM signal with 1 ms period and 25% duty on PB11 (PIN 32 of connector J17)
  • PWM1 generates an active low PWM signal with 10 ms period and 50% duty on PB12 (PIN 33 of connector J17)
  • PWM2 generates an active high PWM signal with 100ms period and 75% duty on PB13 (PIN 31 of connector J 17)


Whenever a counter event occurs on PWM channel 4, console is upated as follows:

  • LED D7 pulses with bluish white hue as all the three PWM signals are connected to the cathodes of the RGB LED.
