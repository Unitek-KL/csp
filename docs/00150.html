Library Interface
a) Initialization Function
 
Name 
Description 
 
ADC_Initialize 
Initializes given instance of ADC peripheral. 
b) Setup Functions
 
Name 
Description 
 
ADC_CallbackRegister 
Registers the function to be called from interrupt. 
 
ADC_ChannelSelect 
Selects ADC input channel. 
 
ADC_ComparisonWindowSet 
Returns the status of automatic sequence conversion. 
c) Transaction Functions
 
Name 
Description 
 
ADC_Enable 
Enables given instance of ADC peripheral. 
 
ADC_Disable 
Disables given instance of ADC peripheral. 
 
ADC_ConversionStart 
Starts the ADC conversion with the software trigger. 
 
ADC_ConversionResultGet 
Returns the conversion result of the channel. 
d) Status Functions
 
Name 
Description 
 
ADC_ConversionStatusGet 
Returns the status of the conversion of the channel. 
 
ADC_WindowMonitorStatusGet 
Returns the status of the window monitor flag. 
e) Data Types and Constants
 
Name 
Description 
 
ADC_CALLBACK 
Defines the data type and function signature for the adc peripheral callback function. 
 
ADC_NEGINPUT 
Identifies ADC negative input pin to select. 
 
ADC_POSINPUT 
Identifies ADC positive input pin to select. 
 
ADC_STATUS 
Identifies ADC interrupt flag status
