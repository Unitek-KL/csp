Library Interface
a) Initialization Function
 
Name 
Description 
 
TCx_CHy_TimerInitialize 
Initializes given instance of TC channel 
b) Setup Functions
 
Name 
Description 
 
TCx_CHy_TimerCallbackRegister 
Registers the function to be called from interrupt 
c) Transaction Functions
 
Name 
Description 
 
TCx_CHy_TimerStart 
Starts the given TC channel counter 
 
TCx_CHy_TimerStop 
Stops the given TC channel counter 
 
TCx_CHy_TimerCompareSet 
Sets the compare value of a given timer channel. 
 
TCx_CHy_TimerCounterGet 
Reads the timer channel counter value 
 
TCx_CHy_TimerFrequencyGet 
Provides the given timer's counter-increment frequency. 
 
TCx_CHy_TimerPeriodSet 
Sets the period value of a given timer channel. 
 
TCx_CHy_TimerPeriodGet 
Reads the period value of given timer channel 
d) Status Functions
 
Name 
Description 
 
TCx_CHy_TimerPeriodHasExpired 
Checks whether timer period is elapsed 
e) Data Types and Constants
 
Name 
Description 
 
TCx_TIMER_CALLBACK 
Defines the function pointer data type and function signature for the tc channel callback function. 
 
TCx_TIMER_CALLBACK_OBJECT 
Callback structure 
 
TCx_TIMER_STATUS 
Identifies channel interrupt source mask
