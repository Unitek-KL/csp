SDHC_CommandErrorGet Function

Returns the errors associated with a command transfer. For a command transfer, the returned value is a bit wise OR of the following errors, Command Timeout Error, Command CRC Error, Data CRC Error and Command End Bit Error. Refer the data sheet for the bit position of the errors.

C
uint16_t SDHC_CommandErrorGet();
Preconditions

A command must have been initiated using the SDHC_CommandSend() function.

Returns

Bit wise OR of the following command error flags - Command Timeout Error, Command CRC Error and Command End Bit Error. Refer the data sheet for the bit position of the errors.

Remarks

None.

Example
uint16_t cmd_error = SDHC_CommandErrorGet();

if (cmd_error != 0)
{
    // Handle command error
}
