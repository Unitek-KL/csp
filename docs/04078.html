Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
RTT_Initialize 
Initialize the RTT peripheral 
b) Setup Functions
 
Name 
Description 
 
RTT_CallbackRegister 
Sets the pointer to the function (and it's context) to be called. 
c) Transaction Functions
 
Name 
Description 
 
RTT_Enable 
Enables the Real Time Timer 
 
RTT_Disable 
Disables the real Time Timer 
 
RTT_AlarmValueSet 
Sets the Alarm value for the RTT peripheral. 
 
RTT_FrequencyGet 
Returns the current RTT frequency 
 
RTT_PrescalarUpdate 
Updates the prescalar for the RTT peripheral. 
 
RTT_TimerValueGet 
Returns the current timer value 
 
RTT_EnableInterrupt 
Enable RTT interrupts 
 
RTT_DisableInterrupt 
Disable RTT interrupts 
d) Data Types and Constants
 
Name 
Description 
 
RTT_CALLBACK 
Defines the data type and function signature for the RTT peripheral callback function. 
 
RTT_INTERRUPT_TYPE 
Defines the data type for the RTT INTERRUPT TYPE. 
 
RTT_OBJECT 
Defines the data type for the RTT OBJECT.
