Library Interface

This section describes the Application Programming Interface (API) functions of the SPI Peripheral Library. 

Refer to each section for a detailed description.

a) Initialization Function
 
Name 
Description 
 
SPIx_Initialize 
Initializes SPI x module of the device 
b) Setup Functions
 
Name 
Description 
 
SPIx_TransferSetup 
Setup SPI operational parameters as desired by the client. 
 
SPIx_CallbackRegister 
Allows application to register callback with PLIB. 
c) Transaction Functions
 
Name 
Description 
 
SPIx_Read 
Read data on SPI x peripheral 
 
SPIx_Write 
Write data on SPI x peripheral 
 
SPIx_WriteRead 
Write and Read data on SPI x peripheral 
d) Status Functions
 
Name 
Description 
 
SPIx_IsBusy 
Returns transfer status of SPI x 
e) Data Types and Constants
 
Name 
Description 
 
SPIx_CALLBACK 
Pointer to a SPI Callback function.
