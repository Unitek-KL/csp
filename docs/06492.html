SDMMC_RESET_TYPE Enumeration

SDMMC error reset type 

The enumeration is used to select the error type to reset.

C
typedef enum {
  SDMMC_RESET_ALL = 0x01,
  SDMMC_RESET_CMD = 0x02,
  SDMMC_RESET_DAT = 0x04
} SDMMC_RESET_TYPE;
Remarks

None.
