CAN_TX_EVENT_FIFO_ELEMENT Structure

CAN Trasmit Event FIFO Element 

This data structure defines the CAN Tx Event FIFO Element format.

C
typedef struct {
  uint32_t te0;
  uint32_t te1;
  uint8_t timestamp[];
} CAN_TX_EVENT_FIFO_ELEMENT;
Members
Members 
Description 
uint32_t te0; 
CAN Tx Event FIFO - SID[10:0] and EID[28:11 
uint32_t te1; 
CAN Tx Event FIFO - DLC[3:0], IDE[4], RTR[5], BRS[6], FDF[7 ESI[8] and SEQ[31:9 
uint8_t timestamp[]; 
CAN Tx Event FIFO - Transmit Message Time Stamp 
Remarks

None.
