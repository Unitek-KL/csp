CLOCK_Initialize Function

This function initializes the clock as defined by the MHC and Clock Manager selections. 

For each of the clock sources enabled in MHC, the function will configure the clock settings and will then wait till the clock is ready. In case of DPLL, the function will wait till a lock is obtained. 

The function will then configure the Generic clock generators and the peripheral bridges based on the MHC configurations.

C
void CLOCK_Initialize();
Preconditions

MHC GUI should be configured with the right values. Incorrect configuration of the Clock will result in incorrect peripheral behavior or a non functional device.

Returns

None.

Example
    CLOCK_Initialize();
