Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
SDHC_Initialize 
Initializes the SDHC peripheral and the internal data structures used by the peripheral library. 
 
SDHC_ModuleInit 
Initializes the SDHC peripheral. 
b) Setup Functions
 
Name 
Description 
 
SDHC_BlockCountSet 
Sets the number of blocks to transfer. 
 
SDHC_BlockSizeSet 
Sets the size of the one data block of transfer. 
 
SDHC_BusWidthSet 
Configures the width of the data bus. 
 
SDHC_CallbackRegister 
This function allows a SDHC PLIB client to set an event handler. 
 
SDHC_DmaSetup 
Sets up the DMA for data tranfers. 
 
SDHC_SpeedModeSet 
Sets the bus speed. 
 
SDHC_CardDetectDisable 
Disables the use of SD Card Detect Pin. 
 
SDHC_CardDetectEnable 
Enables the use of SD Card Detect Pin. 
 
SDHC_ClockSet 
Sets the SDHC clock frequency. 
 
SDHC_ClockEnable 
Enable SDHC clock. 
 
SDHC_ClockDisable 
Disable SDHC clock. 
 
SDHC_WriteProtectDisable 
Disables the use of SD Card Write Protect Pin. 
 
SDHC_WriteProtectEnable 
Enables the use of SD Card Write Protect Pin. 
c) Transaction Functions
 
Name 
Description 
 
SDHC_CommandSend 
This function allows the client to send a SD command on the SDHC interface. 
 
SDHC_ResponseRead 
Reads the response to a given command. 
d) Status Functions
 
Name 
Description 
 
SDHC_DataErrorGet 
Returns the errors associated with a data transfer. 
 
SDHC_CommandErrorGet 
Returns the errors associated with a command transfer. 
 
SDHC_IsCmdLineBusy 
Returns the status of the command line. 
 
SDHC_IsDatLineBusy 
Returns the status of the data line. 
 
SDHC_IsWriteProtected 
Returns the write protect switch pin level. 
 
SDHC_IsCardAttached 
Indicates if the card is attached or detached. 
 
SDHC_ErrorReset 
Resets errors as specified by the resetType. 
e) Data Types and Constants
 
Name 
Description 
 
SDHC_CMD_RESP_TYPE 
The enumeration lists the supported response types 
 
SDHC_BUS_WIDTH 
The enumeration lists the bus widths for the SDHC bus. 
 
SDHC_CALLBACK 
The prototype of the SDHC callback function 
 
SDHC_CLK_MODE 
The enumeration lists the clock modes. 
 
SDHC_DATA_TRANSFER_DIR 
The enumeration lists the direction of the data transfer. 
 
SDHC_DATA_TRANSFER_TYPE 
The enumeration lists single or multi block data transfer type. 
 
SDHC_DataTransferFlags 
The data structure is used by the client to provide information about the data trasnfer to the SDHC PLIB. 
 
SDHC_READ_RESPONSE_REG 
The enumeration lists the response registers that can be read for a given command. 
 
SDHC_RESET_TYPE 
The enumeration lists error type to reset. 
 
SDHC_SPEED_MODE 
The enumeration lists the bus widths for the SDHC bus. 
 
SDHC_XFER_STATUS 
The enumeration lists the status of the transfer. 
 
SDHC_CLOCK_FREQ_400_KHZ 
This is macro sdhc_00187::SDHC_CLOCK_FREQ_400_KHZ. 
 
SDHC_CLOCK_FREQ_DS_25_MHZ 
This is macro sdhc_00187::SDHC_CLOCK_FREQ_DS_25_MHZ. 
 
SDHC_CLOCK_FREQ_HS_50_MHZ 
This is macro sdhc_00187::SDHC_CLOCK_FREQ_HS_50_MHZ.
