PDECx_QDECStatusGet Function

This function returns the status of the events that occur in quadrature mode.

C
PDEC_QDEC_STATUS PDECx_QDECStatusGet();
Preconditions

PDECx_QDECInitialize must have been called first.

Returns

PDEC_QDEC_STATUS - status of the quadrature events

Remarks

This function is applicable only for quadrature mode.

Example
PDEC_QDEC_STATUS status;
status = PDEC_QDECStatusGet();
