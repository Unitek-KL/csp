Running The Application
  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build and Program the application using MPLAB X IDE.
  4. The console displays the ADC Count and the ADC Input Voltage.

  1. Press the switch to change the DAC output by 0.1 V. Observe the ADC input value and the respective ADC count value.
