Running The Application

MPLAB X IDE

  1. Build and Program the application on both boards using the MPLAB X IDE.

Note: Do not execute below Step-2 and Step-3 for PIC32MX1/2/5 Starter Kit and PIC32 Ethernet Starter Kit II as UART is not available for these boards.

  1. Open the Terminal application (Ex.:Tera term) on the computer
  2. Connect to the DEBUG UART Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bit
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Execute the steps and observe the output as described in the following table
Project Name 
Output 
sam_c21n_xpro.X 

  • Press '2' to receive message from CAN Bus on terminal window of SAM C21N XPRO - 2 board.
  • Press '1' to transmit message to CAN Bus on terminal window of SAM C21N XPRO - 1 board.
  • Transmitting Message:Success will be displayed in the terminal window of SAM C21N XPRO - 1 board. CAN message will be displayed in the terminal window of SAM C21N XPRO - 2 board.

 

sam_e54_xpro.X 

  • Press '2' to receive message from CAN Bus on terminal window of SAM E54 XPRO - 2 board.
  • Press '1' to transmit message to CAN Bus on terminal window of SAM E54 XPRO - 1 board.
  • Transmitting Message:Success will be displayed in the terminal window of SAM E54 XPRO - 1 board. CAN message will be displayed in the terminal window of SAM E54 XPRO - 2 board.

 

pic32mk_gp_db.X 

  • Press '1' to transmit message to CAN Bus on terminal window of PIC32MK GP DB board.
  • Transmitting Message:Success will be displayed in the terminal window of PIC32MK GP DB board board. CAN message will be displayed in the CAN analyzer software.
  • Press '2' to receive message from CAN Bus on terminal window of PIC32MK GP DB board.
  • Send following CAN message from CAN analyzer software

ID : 0x469, DLC : 8, Data: 1, 2, 3, 4, 5, 6, 7, 8

  • CAN message will be displayed in the terminal window of PIC32MK GP DB board
  • LED1 toggles at each message transmission or reception

 

pic32mx_125_sk.X 

  • Press Switch 'SW1' to transmit message to CAN Bus on terminal window of PIC32MX 1/2/5 SK board.
  • CAN message will be displayed in the CAN analyzer software.
  • Send following CAN message from CAN analyzer software

ID : 0x469, DLC : 8, Data: 1, 2, 3, 4, 5, 6, 7, 8

  • LED1 toggles at each message transmission or reception

 

pic32mx_eth_sk2.X 

  • Press Switch 'SW1' to transmit message to CAN Bus on terminal window of PIC32MX ETH SK II board.
  • CAN message will be displayed in the CAN analyzer software.
  • Send following CAN message from CAN analyzer software

ID : 0x469, DLC : 8, Data: 1, 2, 3, 4, 5, 6, 7, 8

  • LED1 toggles at each message transmission or reception

 

pic32mz_ef_sk.X 

  • Press '1' to transmit message to CAN Bus on terminal window of PIC32MZ EF SK board.
  • Transmitting Message:Success will be displayed in the terminal window of PIC32MZ EF SK board board. CAN message will be displayed in the CAN analyzer software.
  • Press '2' to receive message from CAN Bus on terminal window of PIC32MZ EF SK board.
  • Send following CAN message from CAN analyzer software

ID : 0x469, DLC : 8, Data: 1, 2, 3, 4, 5, 6, 7, 8

  • CAN message will be displayed in the terminal window of PIC32MZ EF SK board
  • LED1 toggles at each message transmission or reception

IAR Embedded work bench

  1. Open the Terminal application (Ex.:Tera term) on the computer for each SAM 9X60 Evaluation Kit board.
  2. Connect to the USB to DBGU COM port and configure the serial settings for each SAM 9X60 Evaluation Kit board as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build the project using IAR Embedded Workbench
  4. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup for each SAM 9X60 Evaluation Kit board
  5. Execute the steps and observe the output as described in the following table
Project Name 
Output 
sam_9x60_ek.IAR 

  • Press '2' to receive message from CAN Bus on terminal window of SAM 9X60 EK - 2 board.
  • Press '1' to transmit message to CAN Bus on terminal window of SAM 9X60 EK - 1 board.
  • Transmitting Message:Success will be displayed in the terminal window of SAM 9X60 EK - 1 board. CAN message will be displayed in the terminal window of SAM 9X60 EK - 2 board.
  • RGB_LED (Green) toggles at each message transmission or reception
