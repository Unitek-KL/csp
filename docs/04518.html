Running The Application
  1. Build and Program the application using the MPLAB X IDE.
  2. Observe the outputs as described in the following table:

 

Project Name 
Output 
pic32mk_gpl_curiosity_pro.X 
Success: Observe waveforms on oscilloscope.

  • Observe OC1 output RPA0 "Pin 1 of J300" on the oscilloscope which generates an active low output
  • Observe OC2 output RPA14 "Pin 11 of J302" on the oscilloscope which generates an active high output
  • Observe OC3 output RPB0 "Pin 6 of J300" on the oscilloscope which generates the toggle output

  • Observe OC3 output RPB14 "Pin 7 of J501" on the oscilloscope which generates the toggle output
