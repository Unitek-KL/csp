Library Interface
a) Initialization Function
 
Name 
Description 
 
MCPWM_Initialize 
Initializes given instance of MCPWM peripheral 
b) Setup Functions
 
Name 
Description 
 
MCPWM_CallbackRegister 
Registers the function to be called from interrupt 
 
MCPWM_SecondaryEventCallbackRegister 
Registers the function to be called from interrupt 
 
MCPWM_PrimaryEventCallbackRegister 
Registers the function to be called from interrupt 
 
MCPWM_ChannelDeadTimeSet 
Writes dead time values of given MCPWM channel. 
 
MCPWM_ChannelPrimaryDutySet 
Writes the primary duty cycle value of given MCPWM channel 
 
MCPWM_ChannelPrimaryTriggerSet 
Writes the primary trigger compare value to trigger ADC conversion 
 
MCPWM_ChannelSecondaryDutySet 
Writes the secondary duty cycle value of given MCPWM channel 
 
MCPWM_ChannelSecondaryTriggerSet 
Writes the secondary trigger compare value to trigger ADC conversion 
 
MCPWM_PrimaryPeriodSet 
Sets the period value of the primary time base 
 
MCPWM_SecondaryPeriodSet 
Sets the period value of the secondary time base 
c) Transaction Functions
 
Name 
Description 
 
MCPWM_Start 
Starts the MCPWM module 
 
MCPWM_Stop 
Stops the given MCPWM module 
d) Status Functions
 
Name 
Description 
 
MCPWM_PrimaryPeriodGet 
Reads the period value of primary time base generator 
 
MCPWM_SecondaryPeriodGet 
Reads the period value of secondary time base generator 
e) Data Types and Constants
 
Name 
Description 
 
MCPWM_CALLBACK 
Defines the data type and function signature for the MCPWM peripheral callback function for primary event and secondary event 
 
MCPWM_CH_CALLBACK 
Defines the data type and function signature for the MCPWM peripheral callback function 
 
MCPWM_CH_NUM 
Identifies MCPWM channel number in a MCPWM peripheral 
 
MCPWM_CH_STATUS 
Identifies MCPWM channel event status
