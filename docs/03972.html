Description

Description

c) Transaction Functions
 
Name 
Description 
 
PIT_ClearInterrupt 
PIT Clear Interrupt. 
 
PIT_DelayMs 
Delays processing for x milliseconds. 
 
PIT_TimerCompareSet 
Set the timer comparison value. 
 
PIT_TimerPeriodHasExpired

Set the timer period value.

PIT_TimerStart

Description
