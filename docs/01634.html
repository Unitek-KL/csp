DAC_IsReady Function

This function checks for the readiness of the DAC to receive the next sample value. The application should call the DAC_DataWrite() function only when this function returns true. Calling the DAC_DataWrite() function when this function returns false will result in in-determinate operation.

C
bool DAC_IsReady();
Preconditions

DAC_Initialize must have been called for the associated DAC instance.

Returns

true - Ready for receiving next sample value for conversion. false - DAC is BUSY. Not ready for receiving next sample value.

Remarks

None.

Example
    DAC_Initialize();
    if (DAC_IsReady())
    {
        DAC_DataWrite(0xFF);
    }
    else
    {
        // Not ready to accept new conversion request.
        // User Application code.
    }
