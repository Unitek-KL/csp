CN_PIN Enumeration

Change Notice Pins 

This enumeration identifies the available Change Notice pins.

C
typedef enum {
  CN0_PIN,
  CN1_PIN,
  CN2_PIN,
  CN3_PIN,
  CN4_PIN,
  CN5_PIN,
  CN6_PIN,
  CN7_PIN,
  CN8_PIN,
  CN9_PIN,
  CN10_PIN,
  CN11_PIN,
  CN12_PIN,
  CN13_PIN,
  CN14_PIN,
  CN15_PIN,
  CN16_PIN,
  CN17_PIN,
  CN18_PIN,
  CN19_PIN,
  CN20_PIN,
  CN21_PIN
} CN_PIN;
Remarks

The caller should not rely on the specific numbers assigned to any of these values as they may change from one processor to the next. 

Not all pins are available on all devices. Refer to the specific device data sheet to determine which pins are supported.
