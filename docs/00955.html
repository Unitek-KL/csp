Running The Application

MPLAB X IDE

  1. Build and Program the application on both boards using the MPLAB X IDE.
  2. Open the Terminal application (Ex.:Tera term) on the computer for each board
  3. Connect to the DEBUG UART Virtual COM port and configure the serial settings for each board as follows:
    • Baud : 115200
    • Data : 8 Bit
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  4. Execute the steps and observe the output as described in the following table
Project Name 
Output 
sam_c21n_xpro.X 

  • Press '0' to receive message from CAN Bus on terminal window of SAM C21N XPRO - 2 board.
  • Press '2' to transmit message to CAN Bus on terminal window of SAM C21N XPRO - 1 board.
  • Transmitted message description with success will be displayed in the terminal window of SAM C21N XPRO - 1 board. CAN message will be displayed in the terminal window of SAM C21N XPRO - 2 board.
  • Repeat above steps with following input value and observe the output on each terminal.
    • Press '1' on SAM C21N XPRO - 2 board and Press '3' on SAM C21N XPRO - 1 board
    • Press '4' on SAM C21N XPRO - 2 board and Press '6' on SAM C21N XPRO - 1 board
    • Press '5' on SAM C21N XPRO - 2 board and Press '7' on SAM C21N XPRO - 1 board
    • Press '1' on SAM C21N XPRO - 2 board and Press 'a' on SAM C21N XPRO - 1 board
