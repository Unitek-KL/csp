Library Interface
a) Initialization Function
 
Name 
Description 
 
ICAPx_Initialize 
Initializes ICAPx module of the device. 
b) Setup Functions
 
Name 
Description 
 
ICAPx_CallbackRegister 
Sets the callback function for a ICAPx interrupt. 
 
ICAPx_Error_CallbackRegister 
Sets the callback function for a ICAP error interrupt. 
 
ICAPx_Disable 
Disable function for the ICAPx peripheral 
 
ICAPx_Enable 
Enable function for the ICAPx peripheral 
c) Transaction Functions
 
Name 
Description 
 
ICAPx_CaptureBufferRead 
Read buffer function ICAPx peripheral 
d) Status Functions
 
Name 
Description 
 
ICAPx_ErrorStatusGet 
Reads current state overflow status ICAPx status. 
 
ICAPx_CaptureStatusGet 
Reads current state buffer not empty status. 
e) Data Types and Constants
 
Name 
Description 
 
ICAP_CALLBACK 
Defines the function pointer data type and function signature for the ICAPx callback function. 
 
ICAP_STATUS_SOURCE 
Identifies ICAP status source mask.
