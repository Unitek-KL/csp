Library Interface
a) Setup Functions
 
Name 
Description 
 
EFC_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the operation is complete. 
b) Transaction Functions
 
Name 
Description 
 
EFC_RegionLock 
Locks a Flash region. 
 
EFC_RegionUnlock 
Unlocks a Flash region. 
 
EFC_SectorErase 
Erases a Sector in the FLASH. 
 
EFC_PageWrite 
Writes data of size equivalent to page size to a given FLASH address. 
 
EFC_QuadWordWrite 
Writes a 128-bit data to a given address in FLASH memory. 
 
EFC_Read 
Reads length number of bytes from a given address in FLASH memory. 
c) Status Functions
 
Name 
Description 
 
EFC_IsBusy 
Returns the current status of EFC controller. 
 
EFC_ErrorGet 
Returns the error encountered by EFC controller. 
d) Data Types and Constants
 
Name 
Description 
 
EFC_CALLBACK 
Defines the data type and function signature for the EFC peripheral callback function. 
 
EFC_ERROR 
Defines the data type for the EFC Error.
