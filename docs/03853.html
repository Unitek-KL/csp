BACKUP_REGISTER Enumeration

RTC Backup Register Mask 

This enumeration defines the possible Backup Register mask.

C
typedef enum {
  BACKUP_REGISTER_0,
  BACKUP_REGISTER_1,
  BACKUP_REGISTER_2,
  BACKUP_REGISTER_3,
  BACKUP_REGISTER_4,
  BACKUP_REGISTER_5,
  BACKUP_REGISTER_6,
  BACKUP_REGISTER_7
} BACKUP_REGISTER;
Remarks

This Feature may not be available on all devices. Refer to device Datasheet for more information.
