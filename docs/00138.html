Running The Application

MPU Devices

  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the USB_UART Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build the IAR or MPLAB X project using IAR Embedded Workbench or MPLAB X IDE respectively
  4. Use the output binary file (harmony.bin) to setup the board as detailed in the hardware setup
  5. Observe the output as described in the following table
Project Name 
Output 
sam_a5d2_xult.IAR OR sam_a5d2_xult_xc32.X 
The console displays the ADC Count and the ADC Input Voltage.
 
sam_9x60_ek.IAR 
The console displays the ADC Count and the ADC Input Voltage.

    • Stop : 1 Bit
    • Flow Control : None
