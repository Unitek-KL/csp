HSMCI_ERROR_FLAGS Enumeration

HSMCI error flags 

The enumeration is used to identify the error in command and data transfer. The application should use the enumerators in the enum-list to identify the error flags.

C
typedef enum {
  HSMCI_CMD_TIMEOUT_ERROR = 0x0001,
  HSMCI_CMD_CRC_ERROR = 0x0002,
  HSMCI_CMD_END_BIT_ERROR = 0x0004,
  HSMCI_CMD_INDEX_ERROR = 0x0008,
  HSMCI_DATA_TIMEOUT_ERROR = 0x0010,
  HSMCI_DATA_CRC_ERROR = 0x0020,
  HSMCI_DATA_UNDERRUN_ERROR = 0x0040,
  HSMCI_DATA_OVERRUN_ERROR = 0x0080
} HSMCI_ERROR_FLAGS;
