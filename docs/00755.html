OSCCTRL_CallbackRegister Function

This function registers the callback function to be called when the External Oscillator has failed.

C
void OSCCTRL_CallbackRegister(
    OSCCTRL_CFD_CALLBACK callback, 
    uintptr_t context
);
Preconditions

The External Oscillator and/or should have been configured in MHC and enabled.

Parameters
Parameters 
Description 
callback 
Pointer to the OSCCTRL_CFD_CALLBACK type of function that will be called when the oscillator failed.
 
context 
Context value to be passed into the callback function when it called. 
Returns

None.

Remarks

None.
