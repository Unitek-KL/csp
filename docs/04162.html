Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
SDMMCx_Initialize 
Initializes the SDMMC peripheral and the internal data structures used by the peripheral library. 
 
SDMMCx_ModuleInit 
Initializes the SDMMC peripheral. 
b) Setup Functions
 
Name 
Description 
 
SDMMCx_BlockCountSet 
Sets the number of blocks to transfer. 
 
SDMMCx_BlockSizeSet 
Sets the size of the one data block of transfer. 
 
SDMMCx_BusWidthSet 
Configures the width of the data bus. 
 
SDMMCx_CallbackRegister 
This function allows a SDMMC PLIB client to set an event handler. 
 
SDMMCx_ClockDisable 
Disable SDMMC clock. 
 
SDMMCx_ClockEnable 
Enable SDMMC clock. 
 
SDMMCx_ClockSet 
Sets the SDMMC clock frequency. 
 
SDMMCx_DmaSetup 
Sets up the DMA for data tranfers. 
 
SDMMCx_SpeedModeSet 
Sets the bus speed. 
 
SDMMCx_ErrorReset 
Resets errors as specified by the resetType. 
c) Transaction Functions
 
Name 
Description 
 
SDMMCx_CommandSend 
This function allows the client to send a SD command on the SDMMC interface. 
 
SDMMCx_ResponseRead 
Reads the response to a given command. 
d) Status Functions
 
Name 
Description 
 
SDMMCx_CommandErrorGet 
Returns the errors associated with a command transfer. 
 
SDMMCx_DataErrorGet 
Returns the errors associated with a data transfer. 
 
SDMMCx_IsCardAttached 
Indicates if the card is attached or detached. 
 
SDMMCx_IsCmdLineBusy 
Returns the status of the command line. 
 
SDMMCx_IsDatLineBusy 
Returns the status of the data line. 
 
SDMMCx_IsWriteProtected 
Returns the write protect switch pin level. 
 
SDMMCx_ErrorGet 
Returns the errors associated with the peripheral. 
e) Data Types and Constants
 
Name 
Description 
 
SDMMC_BUS_WIDTH 
The enumeration lists the bus widths for the SDMMC bus. 
 
SDMMC_CALLBACK 
The prototype of the SDMMC callback function 
 
SDMMC_CLK_MODE 
The enumeration lists the clock modes. 
 
SDMMC_CMD_RESP_TYPE 
The enumeration lists the supported response types 
 
SDMMC_DATA_TRANSFER_DIR 
The enumeration lists the direction of the data transfer. 
 
SDMMC_DATA_TRANSFER_TYPE 
The enumeration lists single or multi block data transfer type. 
 
SDMMC_DataTransferFlags 
The data structure is used by the client to provide information about the data trasnfer to the SDMMC PLIB. 
 
SDMMC_READ_RESPONSE_REG 
The enumeration lists the response registers that can be read for a given command. 
 
SDMMC_RESET_TYPE 
The enumeration lists error type to reset. 
 
SDMMC_SPEED_MODE 
The enumeration lists the bus widths for the SDMMC bus. 
 
SDMMC_XFER_STATUS 
The enumeration lists the status of the transfer. 
 
SDMMC_CLOCK_FREQ_400_KHZ

SDMMC_CLOCK_FREQ_HS_50_MHZ 
This is macro sdmmc_44002::SDMMC_CLOCK_FREQ_HS_50_MHZ. 
 
SDMMC_CLOCK_FREQ_HS_52_MHZ 
This is macro sdmmc_44002::SDMMC_CLOCK_FREQ_HS_52_MHZ.
