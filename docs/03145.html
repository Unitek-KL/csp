Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
NVMCTRL_Initialize 
Initializes given instance of the NVMCTRL peripheral. 
b) Setup Functions
 
Name 
Description 
 
NVMCTRL_MainCallbackRegister 
Sets the pointer to the function (and it's context) to be called when an operation on the Flash is complete, provided the corresponding interrupt is enabled. 
 
NVMCTRL_SmartEEPROMCallbackRegister 
Sets the pointer to the function (and it's context) to be called when an operation on the SmartEEPROM is complete, provided the corresponding interrupt is enabled. 
c) Transaction Functions
 
Name 
Description 
 
NVMCTRL_BankSwap 
Swaps NVM Banks. 
 
NVMCTRL_BlockErase 
Erases a Block in the NVM 
 
NVMCTRL_DisableMainFlashInterruptSource 
Disables a given interrupt source for the Flash. 
 
NVMCTRL_DisableSmartEepromInterruptSource 
Disables a given interrupt source for the SmartEEPROM. 
 
NVMCTRL_DoubleWordWrite 
Writes two words (8 bytes) into the Flash. 
 
NVMCTRL_EnableMainFlashInterruptSource 
Enables a given interrupt source for the Flash. 
 
NVMCTRL_EnableSmartEepromInterruptSource 
Enables a given interrupt source for the SmartEEPROM. 
 
NVMCTRL_PageWrite 
Writes one page of data to given NVM address. 
 
NVMCTRL_Read 
Reads 'length' number of bytes from a given address in FLASH memory. 
 
NVMCTRL_RegionLock 
Locks an NVMCTRL region. 
 
NVMCTRL_RegionUnlock 
Unlocks a NVM region. 
 
NVMCTRL_SetWriteMode 
Sets the write mode for Flash. 
 
NVMCTRL_SmartEEPROM_IsActiveSectorFull 
Check whether active sector used by the SmartEEPROM is full. 
 
NVMCTRL_SmartEepromFlushPageBuffer 
Flush SmartEEPROM data when in buffered mode. 
 
NVMCTRL_SmartEepromSectorReallocate 
Performs sector reallocation for the SmartEEPROM. 
 
NVMCTRL_QuadWordWrite 
Writes four words (16 bytes) into the Flash. 
d) Status Functions
 
Name 
Description 
 
NVMCTRL_IsBusy 
Checks whether NVMCTRL is ready to perform next command. 
 
NVMCTRL_SmartEEPROM_IsBusy 
Checks whether SmartEEPROM is ready to perform next command. 
 
NVMCTRL_SmartEepromStatusGet 
Returns status conditions of SmartEEPROM. 
 
NVMCTRL_ErrorGet 
Returns error conditions of NVM controller. 
 
NVMCTRL_RegionLockStatusGet 
Returns the value of RUNLOCK register. 
e) Data Types and Constants
 
Name 
Description 
 
NVMCTRL_CALLBACK 
Defines the data type and function signature for the NVMCTRL peripheral callback function. 
 
NVMCTRL_INTERRUPT0_SOURCE 
Defines the Interrupt sources for the main flash. 
 
NVMCTRL_INTERRUPT1_SOURCE 
Defines the Interrupt sources for the SmartEEPROM. 
 
NVMCTRL_WRITEMODE 
Defines the NVMCTRL Write Modes. 
 
NVMCTRL_FLASH_START_ADDRESS 
Defines the start address of NVMCTRL Flash. 
 
NVMCTRL_FLASH_PAGESIZE 
Defines the size (in bytes) of a NVMCTRL Page. 
 
NVMCTRL_FLASH_SIZE 
Defines the size (in bytes) of Flash.
