Running The Application
  1. Build and Program the application using the MPLAB X IDE.
  2. Observe the output as described in the following table:

 

Project Name 
Output 
sam_e70_xult.X 
Success: Observe the waveforms on oscilloscope.

  • Observe PB13 "Pin 5 of the J502" and PD00 "Pin 6 of the J502" on the oscilloscope. Observe a sine wave of 5 KHz.
  • When switch is pressed, waveform changes from sine wave to triangle wave and vice-versa.

 

sam_v71_xult.X 
Success: Observe the waveforms on oscilloscope.

  • Observe PB13 "Pin 5 of the J504" and PD00 "Pin 6 of the J504" on the oscilloscope. Observe a sine wave of 5 KHz.
  • When switch is pressed, waveform changes from sine wave to triangle wave and vice-versa.
