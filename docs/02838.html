Hardware Setup
  1. Project sam_g55_xpro.X
    • Hardware Used
      • SAM G55 Xplained Pro Evaluation Kit
      • EEPROM 3 Click Board
      • mikroBUS Xplained Pro
    • Hardware Setup
      • Plug the EEPROM 3 click Board into the MikroBus socket of the mikroBUS Xplained Pro
      • Connect mikroBUS Xplained Pro board to the EXT1 header of the SAM G55 Xplained Pro Evaluation Kit
      • Connect the Debug USB port on the board to the computer using mini USB cable.

 

  1. Project sam_rh71_ek.X
    • Hardware Used
      • * SAM RH71 Evaluation Kit
      • EEPROM 3 Click Board
      • mikroBUS Xplained Pro
    • Hardware Setup
      • Plug the EEPROM 3 click Board into the MikroBus socket of the mikroBUS Xplained Pro
      • Connect mikroBUS Xplained Pro board to the J24 header of the SAM RH71 Evaluation Kit
      • Connect the debugger probe to J33

 

  1. Project sam_a5d2_xult.IAR OR sam_a5d2_xult_xc32.X
    • Hardware Used
      • SAM A5D2 Xplained Ultra board
      • EEPROM Click 3 Board
      • SD Card with FAT32 file system
    • Hardware Setup
      • Setup the SD card
        • Download harmony MPU bootstrap loader from this location
        • Copy the downloaded boot loader binary( boot.bin) and generated application binary (harmony.bin)into the SD card
        • Insert the SD card into the SDMMC1 slot on the board
      • Connect EEPROM Click 3 Board to XPRO EXT2 (J26) connector on the board as per below Pin Connections table
      • Connect the Debug USB port on the board to the computer using a micro USB cable
    • Pin Connections
XPRO EXT2 (J26) Pins 
EEPROM 4 CLICK Pins 
PIN 11 
SDA 
PIN 12 
SCL 
PIN 19 
GND 
PIN 20 
3V3 

 

  1. Project sam_9x60_ek.IAR
    • Hardware Used
      • SAM 9X60 Evaluation Kit board
      • SD Card with FAT32 file system
    • Hardware Setup
      • Setup the SD card
        • Download harmony MPU bootstrap loader from this location
        • Copy the downloaded boot loader binary( boot.bin) and generated application binary (harmony.bin)into the SD card
        • Insert the SD card into the SDMMC connector J5 on the board
      • Connect the Debug USB port on the board to the computer using a micro USB cable
      • No external hardware is required as on-board EEPROM 24AA025E48 used in demo
      • Note: Reset push button on SAM9X60 EK is labeled as SW3
