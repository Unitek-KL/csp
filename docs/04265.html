SERCOM SPI Applications

This Section provides help for the SERCOM SPI PLIB Applications.

Topics
Name 
Description 
spi_eeprom_write_read 
This example application demonstrates how to use the SERCOM SPI peripheral to write and read from the SPI serial EEPROM memory. 
spi_ping_pong_with_dma 
This example demonstrates how to continuously transmit and receive data over a SPI interface using ping pong buffers with the DMA. 
spi_self_loopback_blocking 
This example application demonstrates how to use the SERCOM SPI peripheral to transmit and receive a block of data in a blocking manner. 
spi_self_loopback_interrupt 
This example application shows how to use SERCOM SPI PLIB with external loop back to write and then read back an array of data in interrupt mode.
