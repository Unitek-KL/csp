Library Interface
a) Initialization Function
 
Name 
Description 
 
ADC_Initialize 
Initializes given instance of ADC peripheral. 
b) Setup Functions
 
Name 
Description 
 
ADC_CallbackRegister 
Registers the function to be called from interrupt 
c) Transaction Functions
 
Name 
Description 
 
ADC_ChannelResultGet 
Reads the conversion result of the channel 
 
ADC_ChannelsEnable 
Enables the ADC channels 
 
ADC_ChannelsDisable 
Disables the ADC channels 
 
ADC_ChannelsInterruptEnable 
Enables the ADC interrupt sources 
 
ADC_ChannelsInterruptDisable 
Disables the ADC interrupt sources 
 
ADC_ComparisonRestart 
Restart the comparison function. 
 
ADC_ComparisonWindowSet 
Sets Low threshold and High threshold in comparison window. 
 
ADC_ConversionSequenceSet 
Sets the user sequence of the channel conversion 
 
ADC_ConversionStart 
Starts the ADC conversion of all the enabled channels with the software trigger 
d) Status Functions
 
Name 
Description 
 
ADC_ChannelResultIsReady 
Returns the status of the channel conversion 
 
ADC_ComparisonEventResultIsReady 
Returns the status of the Comparison event 
e) Data Types and Constants
 
Name 
Description 
 
ADC_CALLBACK 
Defines the function pointer data type and function signature for the adc peripheral callback function. 
 
ADC_CALLBACK_OBJECT 
Callback structure 
 
ADC_CHANNEL_MASK 
Identifies ADC channel mask 
 
ADC_CHANNEL_NUM 
Identifies ADC channel number 
 
ADC_INTERRUPT_MASK 
Identifies interrupt sources number
