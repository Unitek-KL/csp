plib_nvmctrl.h

Microchip Technology Inc.

File Name

plib_nvmctrl.h

Description

Non-Volatile Memory Controller(NVMCTRL) Peripheral Library Interface Header File 

This file defines the interface for the NVMCTRL Plib. It allows user to Program, Erase and lock the on-chip Non Volatile Flash Memory. 

The 'x' will be replaced by the peripheral instance number in the generated headers.
