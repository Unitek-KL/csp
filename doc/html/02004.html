SPIy_DATA_BITS Enumeration

SPI Data Bits 

This enumeration identifies number of bits per SPI transfer.

C
typedef enum {
  SPI_DATA_BITS_8,
  SPI_DATA_BITS_9,
  SPI_DATA_BITS_10,
  SPI_DATA_BITS_11,
  SPI_DATA_BITS_12,
  SPI_DATA_BITS_13,
  SPI_DATA_BITS_14,
  SPI_DATA_BITS_15,
  SPI_DATA_BITS_16
} SPIy_DATA_BITS;

Remarks

For 9 to 15bit modes, data should be right aligned in the 16 bit memory location.
