Running The Application
  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build and Program the application using MPLAB X IDE.
  4. Observe the terminal application. It displays the ADC Count and the ADC Input Voltage.
  5. Press the switch to change the DAC output by 0.1 V.
  6. And observe the changes to the ADC count and the ADC Input Voltage on the terminal application.
  7. Observe the output as described in the following table:
Project Name 
Output 
sam_e70_xult.X 
Success: ADC count and ADC input voltage changes by 0.1 V when switch (SW1) is pressed 
sam_v71_xult.X 
Success: ADC count and ADC input voltage changes by 0.1 V when switch (SW0) is pressed
