SDADCx_ConversionSequenceIsFinished Function

This function checks the status of the conversion sequence.

C
bool SDADCx_ConversionSequenceIsFinished();
Preconditions

SDADCx_Initialize() function must have been called first for the associated instance.And conversion must have been started.

Returns

true - all the enabled channels in the sequence have finished conversion false - sequence conversion is ongoing

Remarks

This function is valid only if auto sequence feature is enabled.

Example
    bool seq_status;
    SDADCx_Initialize();
    SDADCx_ConversionStart();
    seq_status = SDADCx_ConversionSequenceIsFinished();
