Introduction

The Analog-to-Digital Converter (ADC) peripheral supports the following features.

  • 12-bit Resolution with Enhanced Mode up to 14 bits
  • 1 Msps Conversion Rate
  • Digital Averaging Function providing Enhanced Resolution Mode up to 14 bits
  • Selectable Single-ended or Differential Input Voltage
  • Integrated Multiplexers Offering Up to 12 Independent Analog Inputs
  • Individual Enable and Disable of Each Channel
  • Customized channel sequencer to control order of conversion
  • Selectable Conversion Trigger sources
    • External Trigger Pin
    • Timer Counter Outputs
    • ADC Internal Trigger Counter
    • PWM Event Line
    • Software Trigger
    • Continuous mode
  • Sleep Modes
  • Automatic Window Comparison of Converted Values
