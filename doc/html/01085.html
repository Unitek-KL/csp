plib_pio.h

Microchip Technology Inc.

File Name

plib_pio.h

Description

PIO PLIB 

This library provides documentation of all the interface to control and interact with Parallel Input/Output controller (PIO) module.

Enumerations
 
Name 
Description 
 
PIO_PIN 
Identifies the available PIO port pins. 
 
PIO_PORT 
Identifies the available PIO Ports. 
Functions
 
Name 
Description 
 
PIO_Initialize 
Initialize the PIO library. 
 
PIO_PinClear 
Clears the selected pin. 
 
PIO_PinInputEnable 
Enables selected IO pin as input. 
 
PIO_PinInterruptCallbackRegister 
Allows application to register callback for every pin. 
 
PIO_PinInterruptDisable 
Disables IO interrupt on selected IO pin. 
 
PIO_PinInterruptEnable 
Enables IO interrupt on selected IO pin. 
 
PIO_PinLatchRead 
Read the value driven on the selected pin. 
 
PIO_PinOutputEnable 
Enables selected IO pin as output. 
 
PIO_PinRead 
Read the selected pin value. 
 
PIO_PinSet 
Sets the selected pin. 
 
PIO_PinToggle 
Toggles the selected pin. 
 
PIO_PinWrite 
Writes the selected pin. 
 
PIO_PortClear 
Clear the selected IO pins of a port. 
 
PIO_PortInputEnable 
Enables selected IO pins of a port as input. 
 
PIO_PortInterruptDisable 
Disables IO interrupt on selected IO pins of a port. 
 
PIO_PortInterruptEnable 
Enables IO interrupt on selected IO pins of a port. 
 
PIO_PortLatchRead 
Read the latched value on all the I/O lines of the selected port. 
 
PIO_PortOutputEnable 
Enables selected IO pins of a port as output(s). 
 
PIO_PortRead 
Read all the I/O lines of the selected port port. 
 
PIO_PortSet 
Set the selected IO pins of a port. 
 
PIO_PortToggle 
Toggles the selected IO pins of a port. 
 
PIO_PortWrite 
Write the value on the masked I/O lines of the selected port. 
Types
 
Name 
Description 
 
PIO_PIN_CALLBACK 
Pointer to a PIO Pin-Event handler function.
