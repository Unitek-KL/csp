plib_tcc.h

Microchip Technology Inc.

File Name

plib_tcc.h

Description

Timer/Counter for Control(TCC) Peripheral Library Interface Header File 

This file defines the interface to the TCC peripheral library. This library provides access to and control of the associated peripheral instance.

Remarks

This header is for documentation only.
