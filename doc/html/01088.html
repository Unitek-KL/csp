plib_pwm.h

Microchip Technology Inc.

File Name

plib_pwm.h

Description

PWM Peripheral Library Interface Header File 

This file defines the interface to the PWM peripheral library. This library provides access to and control of the associated peripheral instance.

Remarks

This header is for documentation only.
