Library Interface

This section describes the Application Programming Interface (API) functions of the SDADC Library. 

Refer to each section for a detailed description.

a) Initialization Function
 
Name 
Description 
 
SDADCx_Initialize 
Initializes given instance of SDADC peripheral. 
b) Setup Functions
 
Name 
Description 
 
SDADCx_CallbackRegister 
Registers the function to be called from interrupt. 
c) Transaction Functions
 
Name 
Description 
 
SDADCx_ComparisonWindowSet 
Returns the status of automatic sequence conversion. 
 
SDADCx_ConversionResultGet 
Returns the conversion result. 
 
SDADCx_ConversionStart 
Starts the SDADC conversion with the software trigger. 
d) Data Types and Constants
 
Name 
Description 
 
SDADC_CALLBACK 
Defines the function pointer data type and function signature for the SDADC peripheral callback function. 
 
SDADC_CALLBACK_OBJECT 
Callback structure 
 
SDADC_STATUS 
Identifies interrupt status flags 
d) Status Functions
 
Name 
Description 
 
SDADCx_ConversionResultIsReady 
Returns the status of the conversion. 
 
SDADCx_ConversionSequenceIsFinished 
Returns the status of automatic sequence conversion. 
 
SDADCx_WindowMonitorIsSet 
Returns the status of the window monitor.
