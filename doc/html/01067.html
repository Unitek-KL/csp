plib_ac.h

Microchip Technology Inc.

File Name

plib_ac.h

Description

Analog Comparator PLIB 

This file defines the interface to the AC peripheral library. This library provides access to and control of the Analog Comparator.

Enumerations
 
Name 
Description 
 
ACy_CHANNEL 
Identifies the AC output channel number

Description

ACy_Initialize

Triggers a comparator to start comparison 
 
ACy_StatusGet

ACy_SwapInputs 
Swap inputs of a comparator.
