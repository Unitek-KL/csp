TCx_CaptureStop Function

This function stops the clock and thus timer.

C
void TCx_CaptureStop();
Preconditions

TCx_CaptureInitialize() function must have been called first.

Returns

None.

Remarks

None.

Example
  TCx_CaptureInitialize();
  TCx_CaptureStart();
  TCx_CaptureStop();
