plib_freqm.h

Microchip Technology Inc.

File Name

plib_freqm.h

Description

FREQUENCY METER(FREQM) PLIB 

This file defines the interface to the FREQM peripheral library. This library provides access to and control of the associated peripheral instance.

Remarks

This header is for documentation only.
