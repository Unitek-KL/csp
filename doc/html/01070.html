plib_afec.h

Microchip Technology Inc.

File Name

plib_afec.h

Description

AFEC Peripheral Library Interface Header File 

This file defines the interface to the AFEC peripheral library. This library provides access to and control of the associated peripheral instance.

Remarks

This header is for documentation only.
