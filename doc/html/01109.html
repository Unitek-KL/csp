plib_usart.h

Microchip Technology Inc.

File Name

plib_usart.h

Description

USART Peripheral Library Interface Header File 

This file defines the interface to the USART peripheral library. This library provides access to and control of the associated peripheral instance.

Remarks

This header is for documentation only.

Enumerations
 
Name 
Description

Defines the data type for the USART data bit length.

Functions
 
Name 
Description 
 
USARTy_ErrorGet 
Gets the error of the given USART peripheral instance. 
 
USARTy_Initialize 
Initializes given instance of the USART peripheral. 
 
USARTy_Read 
Submits a read buffer to the given USART peripheral to process. 
 
USARTy_ReadByte 
Submits request to read a byte of data to the given USART peripheral. 
 
USARTy_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given USART's read events occur. 
 
USARTy_ReadCountGet 
Returns the count of number of bytes processed for a given USART read operation. 
 
USARTy_ReadIsBusy 
Returns the read request status associated with the given USART peripheral instance. 
 
USARTy_ReceiverIsReady 
Returns the hardware status of the USART Receiver. 
 
USARTy_SerialSetup 
Sets the USART serial communication settings dynamically. 
 
USARTy_TransmitterIsReady 
Returns the hardware status of the USART Transmitter. 
 
USARTy_Write 
Submits a write buffer to the given USART peripheral to transfer. 
 
USARTy_WriteByte

USARTy_WriteCallbackRegister

USARTy_WriteCountGet 
Returns the count of number of bytes processed for a given USART write operation. 
 
USARTy_WriteIsBusy 
Returns the write request status associated with the given USART peripheral instance.
