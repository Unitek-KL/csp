FREQMx_FrequencyGet Function

This function returns the measured frequency in Hz. It should be called when a frequency measurement is complete and no errors have occurred. This function is non-blocking. In this mode, the function should be called only after a callback function was called. The return value of the function is only valid if the FREQMx_ErrorGet() function returns FREQM_ERROR_NONE.

C
uint32_t FREQMx_FrequencyGet();
Preconditions

FREQMx_Initialize() and FREQM_MeasurementStart() functions should be called.

Returns

uint32_t - frequency value of the measured clock. The value is only valid if the FREQMx_ErrorGet() function returns FREQM_ERROR_NONE.

Remarks

None.

Example
// Refer to the code example of FREQMx_CallbackRegister() function for
// guidance on interrupt based operation.

// The following code snippet shows an exampl of non-interrupt based
// operation.

FREQMx_Initialize();
FREQM_MeasurementStart()

// Wait till the measurement is complete.
while(FREQMx_IsBusy());

if(FREQM_ERROR_NONE == FREQMx_ErrorGet())
{
    measuredFrequency = FREQMx_FrequencyGet();
    // This means there are no errors and the value contained in
    // measuredFrequency is valid.
}
