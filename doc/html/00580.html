EICz_PinDebounceStateGet Function

This function gets the De-bounced state of the EIC Pin.

C
bool EICz_PinDebounceStateGet(
    EICz_PIN pin
);
Preconditions

EICz_Initialize() must have been called first for the associated instance. The debounce feature should have been enabled in MHC on the desired pin.

Parameters
Parameters 
Description 
pin 
EIC Pin number 
Returns

true - If the EIC Pin De-bounce state is set. false - If the EIC Pin De-bounce state is not set

Remarks

None.

Example
if (true == EICz_PinDebounceStateGet(EIC_PIN_3))
{
    // The pin was debounced and the state of the pin is logic high.
}
