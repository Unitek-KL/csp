Microchip Support
dmac_memory_transfer

The application uses a software trigger to initiate a memory-memory transfer from the source buffer to the destination buffer with 16-bit beat size and 32-bit beat size. 

SAMC21n, SAMD21 and SAME54: 

The number of cycles taken for the DMA transfer is measured using the System timer (SysTick) and reported in the console. 

PIC32MZ: 

Only transfer status reported in console
