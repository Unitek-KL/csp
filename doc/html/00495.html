EFCx_QuadWordWrite Function

This function takes a 32-bit address, a pointer to the 128-bit data and writes it to the given location in FLASH memory.

C
bool EFCx_QuadWordWrite(
    uint32_t* data, 
    uint32_t address
);
Preconditions

Validate if EFCx controller is ready to accept new request by calling EFCx_IsBusy() 

The Memory to be written should be in Erased State.

Parameters
Parameters 
Description 
address 
FLASH address to be modified 
data 
pointer to 128-bit data buffer 
Returns
  • true: On successfull write request.
  • false: On failure to send write request.
Remarks

Application needs to poll for busy bit or wait for callback to be called before sending next request.

Example
    EFC0_QuadWordWrite((uint32_t *) buffer, 0x500000);
    while(EFC0_IsBusy());
