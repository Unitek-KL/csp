plib_spi.h

Microchip Technology Inc.

File Name

plib_spi.h

Description

SPI PLIB 

This library provides documentation of all the interfaces which can be used to control and interact with an instance of a Serial Peripheral Interface (SPI) controller. This file must not be included in any MPLAB Project.
