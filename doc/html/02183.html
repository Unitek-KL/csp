Running The Application
  1. Build and program the application using the MPLAB X IDE.
  2. Open the Terminal application (Ex.:Tera term) on the computer.
  3. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  4. Reset the device and see the following message in the console. The LED is blinked continuously using the SysTick Timer.
