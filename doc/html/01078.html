plib_efc.h

Microchip Technology Inc.

File Name

plib_efc.h

Description

Interface definition of EFC PLIB. 

This file defines the interface for the EFC Plib. It allows user to Program, Erase and lock the on-chip FLASH memory.

Enumerations
 
Name 
Description 
 
EFC_ERROR 
Defines the data type for the EFC Error. 
Functions
 
Name 
Description 
 
EFCx_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the operation is complete. 
 
EFCx_ErrorGet 
Returns the error encountered by EFCx controller. 
 
EFCx_IsBusy 
Returns the current status of EFCx controller. 
 
EFCx_PageWrite 
Writes data of size equivalent to page size to a given FLASH address. 
 
EFCx_QuadWordWrite 
Writes a 128-bit data to a given address in FLASH memory. 
 
EFCx_Read 
Reads length number of bytes from a given address in FLASH memory. 
 
EFCx_RegionLock 
Locks a Flash region. 
 
EFCx_RegionUnlock 
Unlocks a Flash region. 
 
EFCx_SectorErase 
Erases a Sector in the FLASH.
