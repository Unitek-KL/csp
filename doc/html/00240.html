Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
AFECx_Initialize 
Initializes given instance of AFEC peripheral. 
b) Setup Functions
 
Name 
Description 
 
AFECx_CallbackRegister 
Registers the function to be called from interrupt 
 
AFECx_ChannelGainSet 
Writes the gain of the channel 
 
AFECx_ChannelOffsetSet 
Writes the channel offset 
c) Transaction Functions
 
Name 
Description 
 
AFECx_ChannelResultGet 
Reads the conversion result of the channel 
 
AFECx_ChannelsEnable 
Enables the ADC channels 
 
AFECx_ChannelsDisable 
Disables the ADC channels 
 
AFECx_ChannelsInterruptDisable 
Disables the ADC interrupt sources 
 
AFECx_ChannelsInterruptEnable 
Enables the ADC interrupt sources 
 
AFECx_ConversionSequenceSet 
Sets the user sequence of the channel conversion 
 
AFECx_ConversionStart 
Starts the ADC conversion of all the enabled channels with the software trigger 
d) Status Functions
 
Name 
Description 
 
AFECx_ChannelResultIsReady 
Returns the status of the channel conversion 
e) Data Types and Constants
 
Name 
Description 
 
AFEC_CALLBACK 
Defines the function pointer data type and function signature for the afec peripheral callback function. 
 
AFEC_CALLBACK_OBJECT 
Callback structure 
 
AFEC_CHANNEL_GAIN 
Identifies programmable gain setting 
 
AFEC_CHANNEL_MASK 
Identifies AFEC channel mask 
 
AFEC_CHANNEL_NUM 
Identifies AFEC channel number 
 
AFEC_INTERRUPT_MASK 
Identifies channel interrupt sources mask
